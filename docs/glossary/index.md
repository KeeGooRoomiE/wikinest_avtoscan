# Глоссарий

Словарь терминов и сокращений, используемых в компании автоскан.

| Термин | Расшифровка | Описание |
|--------|-------------|----------|
| ТП | Техническая поддержка | Отдел технической поддержки клиентов |
| ОП | Отдел продаж | Коммерческий отдел компании |
| СО | Сервисный отдел | Отдел монтажа и обслуживания оборудования |
| ОТП | Отдел технической поддержки | Внутреннее название отдела ТП |
| ТС | Транспортное средство | Автомобиль или иная единица транспорта клиента |
| ГЛОНАСС | — | Глобальная навигационная спутниковая система (РФ) |
| GPS | Global Positioning System | Американская навигационная спутниковая система |
| CAN | Controller Area Network | Шина данных автомобиля |
| ТМЦ | Товарно-материальные ценности | Оборудование и материалы на складе |
| ТТН | Товарно-транспортная накладная | Документ для перевозки грузов |
| КП | Коммерческое предложение | Документ с условиями и ценами для клиента |
| УПД | Универсальный передаточный документ | Аналог счёт-фактуры + накладной |
| ЭДО | Электронный документооборот | Обмен документами в электронном виде |
| АТ | — | Адресный терминал / устройство мониторинга |
| НВК | Новокузнецк | Офис автоскан в Новокузнецке |
| КЕМ | Кемерово | Главный офис автоскан |
| ЕКБ | Екатеринбург | Офис автоскан-Екатеринбург |
