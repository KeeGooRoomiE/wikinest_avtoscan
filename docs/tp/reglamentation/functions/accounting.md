# Должностные функции — Бухгалтерия

## Зона ответственности

Бухгалтер обеспечивает ведение бухгалтерского и налогового учёта компании.

## Основные функции

- Ведение бухгалтерского учёта в 1С
- Подготовка и сдача налоговой отчётности
- Работа с первичной документацией (накладные, акты, счета-фактуры)
- Контроль дебиторской и кредиторской задолженности
- Начисление заработной платы
- Работа с банком и платёжными поручениями

## Инструменты

- 1С Бухгалтерия
- Банк-клиент
- Электронный документооборот
