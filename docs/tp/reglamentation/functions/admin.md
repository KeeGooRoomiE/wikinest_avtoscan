# Должностные функции — Администратор

## Зона ответственности

Администратор отвечает за работоспособность базы знаний, настройку прав доступа и поддержку инфраструктуры компании.

## Основные функции

- Управление базой знаний: структура, разделы, права доступа
- Контроль актуальности документации
- Техническая поддержка сотрудников по вопросам инструментов и сервисов
- Администрирование корпоративных сервисов (CRM, 1С, системы мониторинга)

## Инструменты

- База знаний Автоскан
- GitHub (репозиторий проекта)
- Системы мониторинга (Axenta, Wialon, Глонасс Soft)
