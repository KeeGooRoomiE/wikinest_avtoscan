# Должностные функции — Кладовщик

## Зона ответственности

Кладовщик отвечает за приём, хранение и выдачу оборудования и материалов на складе.

## Основные функции

- Приём оборудования и сверка с накладными
- Учёт товарно-материальных ценностей (ТМЦ) в 1С
- Выдача оборудования монтажникам и установщикам
- Контроль остатков на складе
- Участие в инвентаризациях
- Работа с поставщиками (приём поставок)

## Инструменты

- 1С (складской учёт)
- Документация по оборудованию из раздела «Оборудование»
