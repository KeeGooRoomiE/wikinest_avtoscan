# Должностные функции — Техническая поддержка

## Зона ответственности

Специалист технической поддержки обеспечивает подключение, настройку и сопровождение оборудования и сервисов мониторинга у клиентов.

## Основные функции

- Первичная настройка и подключение терминалов (ADM, Arnavi, GalileoSky и др.)
- Создание и настройка объектов в системах мониторинга (Axenta, Wialon, Глонасс Soft)
- Консультация клиентов по работе с системами
- Диагностика и устранение технических неисправностей
- Ведение документации по выполненным работам

## Инструменты

- Системы мониторинга: Axenta, Wialon, Глонасс Soft
- 1С (учётная система)
- Регламенты из раздела «Регламенты» базы знаний
