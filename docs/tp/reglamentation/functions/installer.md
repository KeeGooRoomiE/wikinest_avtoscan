# Должностные функции — Установщик

## Зона ответственности

Установщик выполняет монтаж и настройку оборудования мониторинга на транспортных средствах клиентов.

## Основные функции

- Монтаж GPS/ГЛОНАСС терминалов и сопутствующего оборудования на ТС
- Подключение датчиков (зажигание, топливо, CAN-шина, доп. оборудование)
- Первичная настройка и тестирование терминала после монтажа
- Оформление актов выполненных работ
- Соблюдение регламентов монтажа

## Инструменты

- Регламенты монтажа из раздела «Оборудование»
- Инструкции по терминалам (ADM, Arnavi, GalileoSky, и др.)
