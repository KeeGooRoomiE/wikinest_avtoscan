# Навтелеком

Ссылка на NTC Wiki https://wiki.navtelecom.ru/ru/home/devices/commands/standard_cmd

## Основные команды

| Команда | Описание |
|---|---|
| `RESET` | Перезагрузка |
| `8749_V` | Отклик прибора |
| `8749_*!CNCT_RCS` | Подключение |

## Смена сервера

```
SET 95.163.12.22 20140
```
> Для старых S-2333 — всё через пробел

```
*!EDITS TRANS:SRV1(FLEX,,,193.193.165.165,21626)
*!EDITS TRANS:SRV1(FLEX,,,wialon.avtoscan.com,21626)
*!EDITS TRANS:SRV2(FLEX,,,hw.axenta.cloud,21889)
```

## Прочие команды

| Команда | Описание |
|---|---|
| `*!REP_FL 1,09.10.25,10.10.25` | Перевыгрузить историю за период |
| `*!EDITS SYSTEM:PASS(4206,4206)` | Поменять пароль |
| `*!EDITS SYSTEM:PASS(!,!)` | Удалить пароль |
| `*!CNCT_RFU 89.208.152.55,9100,LAST` | Обновить прошивку |
| `*!SYNC 1` | Синхронизация с сервером 1 |

## Совместимые серии

## Эскорт ТД BLE — коды ошибок

| Значение RS-485/BT | Описание |
|---|---|
| `7000` | Короткое замыкание — скопление воды/грязи в баке. Слить остатки, промыть трубки |
| `6500` | Обрыв внешней и внутренней измерительных трубок |
| `65530`, `0`, `65532`, `-4` | Потеря связи с трекером / нет питания / села батарея. `65530` — код трекеров Навтелеком |
| `65522` (`-14`) | Неверное значение уровня (общий код ошибки LLS) |
