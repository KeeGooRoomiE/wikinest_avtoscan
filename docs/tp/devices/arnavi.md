# Arnavi (A4 / A4MAX)

## Важно

> **A4MAX: слева — GPS, справа — GSM**

## Смена сервера (SMS)

```
<пароль>*#1=95.163.12.22,20144
```
> Пароль по умолчанию — `123456`

Примеры:
```
123456*SETP*#1=wialon.avtoscan.com,20623,0    (первый сервер)
123456*SETP*#2=hw.axenta.cloud,20623,0         (второй сервер)
```
> Второй сервер может не настроиться на всех приборах — пинг есть, но данные не шлёт.

Без пароля:
```
SETP*#1=193.193.165.165,20629,4
```

## Смена пароля

```
123456*SETP*#3=13571    → установить пароль 13571
```

## TCP команды

| Код | Описание |
|---|---|
| `0105` | Обновить ПО |
| `0107` | Рестарт устройства |
| `0108` | Обновить настройки (применить после смены конфигурации) |

## Перевод с сервера (Wialon → Axenta)

```
123456*SETP*#1=wialon.avtoscan.com,<порт>,0
123456*SETP*#2=hw.axenta.cloud,20623,0
```
