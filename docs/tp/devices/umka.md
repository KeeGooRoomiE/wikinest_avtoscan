# УМКа

## УМКа 312 — команды

| Команда | Описание |
|---|---|
| `WHO` | Запросить модель и IMEI терминала |
| `AUTH 8749` | Авторизация номера телефона |
| `REMCFG ENABLE` | Включить удалённое конфигурирование |
| `SETSERV 95.163.12.22:20213` | Смена сервера и порта |
| `RELOAD` | Перезагрузка |
| `UPDATE` | Обновление прошивки (не все терминалы) |

---

## УМКа 310 — менеджер питания

### VOLTSAVE

Переход в режимы энергосбережения по уровню напряжения на аналоговом входе.

```
VOLTSAVE <вход>,<STANDBY_мВ>,<IDLE_мВ>
```

| Параметр | Описание |
|---|---|
| `0` / `1` | Номер аналогового входа |
| `-1` | Бортовое напряжение |
| `STANDBY_мВ` | Порог перехода в режим ожидания (модем отключён, SMS не принимает) |
| `IDLE_мВ` | Порог перехода в режим бездействия |

> В модели 310 нет второго аналогового входа, в 311 — первого. Для этих терминалов параметр входа игнорируется.

### ACTIVEWIN

Окно активности — выводит терминал из энергосбережения в заданное время.

```
ACTIVEWIN <смещение_сек>,<длительность_сек>
```

| Параметр | Описание |
|---|---|
| `смещение_сек` | Секунды от начала суток по UTC |
| `длительность_сек` | Длительность окна. `0` — отключено |

---

## Пример: маяк на УМКа 310

Условия:
- Напряжение < 13,5 В → режим бездействия (IDLE)
- Напряжение < 12,5 В → режим ожидания (STANDBY)
- Раз в сутки в 12:00 МСК (09:00 UTC) → активен 30 минут

```
VOLTSAVE 0,12500,13500
ACTIVEWIN 32400,1800
RELOAD
```

## АКБ — справочно

| Параметр | Значение |
|---|---|
| Макс. напряжение батареи | 4,2 В |
| Отключение при | 3,25 В |
| Диапазон настройки ёмкости | 250–1100 мАч (команда `CHARGE`) |
| Среднее потребление (13,8 В) | 70 мА |
| Максимальное потребление | 200 мА |

> При установке АКБ 2000 мАч с настройкой `CHARGE 1100` — заряд током ~1А, время заряда ~×2. Включить режим быстрой зарядки.
