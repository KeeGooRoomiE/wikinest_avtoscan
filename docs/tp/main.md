# Основная информация для техподдержки

### Основное

Пароль для редактирования документации

``` 
**8160**
```

Вводится при изменении каждого документа


Данные для удаленного входа на **отдельного** пользователя на большинстве машин:
```
sshuser
qwerty123
```
На устройствах может быть прокинуто не на всех, для этого есть **ssh_user.bat** в общей папке сервера
