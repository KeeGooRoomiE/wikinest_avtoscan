# Vega MT / CAN-WAY

## Основные команды

| Команда | Описание |
|---|---|
| `@PIN:reboot` | Рестарт блока |
| `@PIN:info?` | Запросить текущее состояние |
| `@PIN:server?` | Запросить настройки серверов мониторинга |

## Смена сервера

Восстановление доступа к серверу (ответ: `server 3 config is valid`):
```
@1234:server3:89.189.183.233:5604&vega&0&0
```

Перевод на Гелиос:
```
@8749:server1:95.163.12.22:20213&wips&0&0
```

Перевод с сервера (где `yyyyy` — порт на Wialon):
```
@xxxx:server1:wialon.avtoscan.com:yyyyy&wips&0
@xxxx:server2:hw.axenta.cloud:21000&wips&0
```

## PIN-коды по умолчанию

| Команда | PIN |
|---|---|
| `@PIN:reboot` | `3333` |
| `@PIN:info?` | `1234` |
| `@PIN:server?` | `4444` |

## CAN-WAY

```
@1234:server1:5.159.100.97:20199:20199&wips&0&90008
```

| Команда | Описание |
|---|---|
| `info?` | Запросить текущее состояние блока |
| `server?` | Запросить настройки серверов мониторинга |
