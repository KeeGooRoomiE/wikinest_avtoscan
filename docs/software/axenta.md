## Арифметические операции с параметрами датчиков

### Базовые операции

```
param1 + param2
param1 - param2
param1 * param2
param1 / const1000
```

> Пробелы вокруг оператора не обязательны. Для управления порядком операций — скобки: `(param1 + param2) / param3`

### Специальные конструкции

| Конструкция | Описание |
|---|---|
| `const1000` | Константа. Число указывается после `const` |
| `#param1` | Значение параметра из предыдущего сообщения |
| `param1[2]` | Побитовое извлечение бита №2 |
| `[Название датчика]` | Ссылка на другой датчик по имени |
| `get(param1,param2)` | Первый существующий параметр (логический «или») |
| `hex(param1)` | Перевод из DEC в HEX |
| `slice(param1,const1,const2)` | Получение диапазона символов (нумерация с 0) |

### Примеры

2 ДУТ в одном баке:
```
([ДУТ1]+[ДУТ2])/const2
```

2 ДУТ в разных баках:
```
[ДУТ1]+[ДУТ2]
```

Сложение датчиков топлива:
```
[Топливо бак1] + [Топливо бак2]
```

Разница с предыдущим значением:
```
in1 - #in1
```

Slice пример — убрать первые 3 символа из `RUD0000002282403`:
```
slice(param1,const3,const14)    → 00000022824
```

## Перевод IMEI в 10 цифр (протокол EGTS)

1. Открыть Калькулятор → вид «Программист»
2. Вставить IMEI → перевести DEC → HEX
3. Нажать `And`
4. Нажать `F` 8 раз → нажать `DEC` → нажать `=`
