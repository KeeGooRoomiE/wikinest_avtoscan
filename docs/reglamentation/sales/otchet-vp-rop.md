# Инструкция по формированию ежемесячного отчёта по валовой прибыли

*В папку руководителя отдела продаж. 29.03.2024.*

## Цель

Данная инструкция разработана для того, чтобы РОП мог самостоятельно, без ошибок и по установленным правилам формировать отчёт о финансовых показателях отдела за месяц. Эта инструкция не содержит пошаговых действий для формирования отчёта, а акцентирует внимание на тех местах, в которых наиболее часто возникают ошибки при формировании отчёта.

## Показатели, которые необходимо отобразить в отчёте

1. **Выручка, себестоимость, валовая прибыль.** Данные берём из отчёта в 1С «Анализ оплат по заказам клиентов». Как правило выручка в данном отчёте отображается корректно, а себестоимость и валовую прибыль необходимо скорректировать вручную в связи с отклонениями по заказам. Для того чтобы учесть отклонения, необходимо в «Анализе оплат по заказам клиентов» по каждому заказу с низким приоритетом (подсвечиваются жёлтым цветом) изменить себестоимость и ВП в соответствии с теми комментариями, которые указаны в заказе. В течение месяца РОПу необходимо проверять, чтобы менеджеры указывали в комментариях отклонения по заказам и устанавливали этим заказам низкий приоритет. Также из выручки, себестоимости и ВП необходимо вычитать те заказы, которые не связаны с продажами менеджеров, например поступления за арендную плату от компании Актион. После формирования отчёта ВП в нём должна быть равна сумме ВП по каждому МОП.

2. **Количество проданного оборудования.** Для подсчёта количества проданного оборудования переходим в отчёт 1С «Анализ оплат по заказам клиентов» и выбираем через отбор «номенклатура» требуемый тип оборудования. Важно добавить в отбор все наименования оборудования по каждому типу (например, для подсчёта терминалов необходимо выбрать все виды оборудования от Навтелекома, Галилео и Телтоника). Во избежание ошибок необходимо также учесть то оборудование, которое было продано услугами (такие заказы подсвечиваются жёлтым цветом). Если в течение месяца были «перепривязки» или возврат ДС клиенту, то в анализе оплат по данному заказу оборудование будет отображаться со знаком минус — в этом случае расчёт производят вручную, не вычитая из общего числа оборудование, по которому произошёл возврат или «перепривязка». Если в отчётном месяце произошла частичная оплата оборудования, в отчёте отображают то оборудование, по которому получена оплата более 50 %; если оплата 50 % и менее, оборудование не фиксируется в отчётном месяце.

3. **Сумма дебиторской и кредиторской задолженности.** Данные по дебиторской и кредиторской задолженности берём из отчёта в 1С «Задолженность клиентов по менеджерам». В этот отчёт могут попасть клиенты, задолженность/переплата у которых относится к другому филиалу — их нужно исключить вручную, поскольку клиент может одновременно работать с несколькими филиалами.

После заполнения всех полей отчёт отправляется главному бухгалтеру до 10 числа текущего месяца.

## Приложение 1. Форма ежемесячного отчёта по ВП для РОП

Перечень предоставления информации главному бухгалтеру Поповой Т.М. для формирования ежемесячного управленческого баланса. Информация за предыдущий отчётный месяц предоставляется Дербеневым Ю.И. до 10 числа текущего месяца.

**Поступление денежных средств (пример, февраль 2024):**

| Статья | Сумма, руб. |
| --- | --- |
| Выручка | 6 284 862,24 |
| Себестоимость | 3 552 818,52 |
| Валовая прибыль | 2 732 043,72 |

**Продажа оборудования (пример, февраль 2024):**

| Оборудование | Количество, шт. |
| --- | --- |
| Терминалы | 77 |
| Тахографы | 22 (Меркурий — 0, Атол — 21, Штрих — 1) |
| СКЗИ | 70 |
| КВ | 100 |

**Дебиторская задолженность покупателей (пример, февраль 2024):** 3 200 492,20 руб.
**Кредиторская задолженность перед покупателями (пример, февраль 2024):** 3 696 458,94 руб.
