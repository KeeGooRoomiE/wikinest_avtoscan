# СИГНАЛ S-2613

<img src="/docs/assets/devices/nav-terminals/navtelecom/navtelecom-s-2613.png" alt="СИГНАЛ S-2613" style="max-width:400px;height:auto">

СИГНАЛ S-2613 — автомобильный навигационный терминал для спутникового мониторинга ТС, контроля топлива и подключения охранных и управляющих цепей. Устройство определяет координаты по ГЛОНАСС/GPS/BeiDou, передает телематические данные по GSM/GPRS (2G) и использует одну внешнюю miniSIM-карту. Для подключения оборудования предусмотрены четыре дискретных входа, один аналоговый вход, четыре управляющих выхода и интерфейс RS-485. Терминал поддерживает до шести цифровых ДУТ по RS-485, совместимые беспроводные ДУТ и датчики температуры по Bluetooth, а также подключение микрофона и динамика для громкой связи с водителем. Встроенная АКБ 800 мА·ч обеспечивает не менее 6 часов автономной работы при передаче данных один раз в минуту. В линейке СИГНАЛ S-2613 следует выбирать для базовых проектов, где не требуются CAN, RS-232, 1-Wire, microSD, работа с тахографом, EcoDriving и фиксация ДТП. Для проектов с тахографом, CAN-шиной и расширенным набором интерфейсов предназначены модели S-2651 и S-2653. Корпус имеет степень защиты IP54; при работе со встроенной АКБ допустимый температурный диапазон составляет от −20 до +60 °C.

## Сервисное обслуживание

Настройка, управление, передача данных и диагностика терминала выполняются через USB с использованием NTC Configurator. Настройка и управление также поддерживаются по Bluetooth через мобильное приложение NTC Control; прошивка и параметры могут обновляться удаленно по GPRS, предусмотрена работа в системе удаленного управления трекерами DRC. Техническая документация, прошивки и материалы для настройки размещаются на официальном сайте «Навтелеком». Ремонт, техническое обслуживание или модернизация, выполненные лицами, не уполномоченными производителем, приводят к утрате права на гарантийное обслуживание.

## Гарантия

Гарантийный срок на терминал составляет 3 года с даты продажи. На встроенный аккумулятор и батарейку предоставляется отдельная гарантия сроком 1 год, также исчисляемая с даты продажи. В течение гарантийного срока производитель выполняет бесплатный ремонт при соблюдении правил транспортирования, хранения, монтажа и эксплуатации. Право на гарантийное обслуживание утрачивается при несанкционированном ремонте, техническом обслуживании или модернизации, нарушении пломб или стикеров, повреждениях от механических, химических, температурных и иных внешних воздействий, а также при эксплуатации совместно с неисправным или неподтвержденным совместимым оборудованием. Незначительные потертости, не влияющие на технические характеристики, не являются основанием для отказа в гарантии.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Навигационные системы | ГЛОНАСС/GPS/BeiDou |
| Канал передачи данных | GSM/GPRS (2G); Bluetooth 4.0 |
| SIM-карта | 1: внешняя miniSIM с выталкивателем |
| Антенны | Внешние GNSS и GSM, разъемы SMA |
| Погрешность определения координат | не более 2,5 м в плане; 5 м по высоте (50% CEP, статический режим) |
| Встроенная АКБ | Li-Po 3,7 В, 800 мА·ч |
| Автономная работа | не менее 6 ч при передаче данных по GSM один раз в минуту |
| Входные линии | 4 дискретных и 1 аналоговая; аналоговый вход измеряет 0–31 В |
| Цифровые интерфейсы | RS-485; USB для настройки и диагностики |
| Отсутствующие интерфейсы | CAN, RS-232 и 1-Wire не предусмотрены |
| Управляющие выходы | 4 линии типа «открытый коллектор» |
| Внутренняя память | 8 МБ; до 51 700 телематических записей при пакете менее 127 байт |
| Карта памяти | microSD не поддерживается |
| Акселерометр | диапазон ±16 g; датчики ударов, перемещения и наклона |
| Детекторы глушения | GSM и GNSS |
| Степень защиты корпуса | IP54 |
| Рабочая температура со встроенной АКБ | от −20 до +60 °C |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Мониторинг ТС | Местоположение, маршруты, пробег, стоянки, скорость и моточасы |
| Контроль топлива | Один аналоговый ДУТ; по RS-485 — до 6 цифровых ДУТ LLS |
| Беспроводные датчики | По Bluetooth — до 4 совместимых ДУТ и до 4 датчиков температуры и влажности |
| Дискретные сигналы | Контроль зажигания, тревожной кнопки, дверей, капота и других контактных датчиков |
| Связь с водителем | Проводное подключение микрофона и динамика для громкой связи и микрофонного прослушивания |
| Охранные функции | Режим охраны, контроль эвакуации, тревожные события, детекторы глушения GSM и GNSS |
| Удаленное управление | Четыре выхода для сирены, реле блокировки и других исполнительных устройств |
| Работа при отсутствии связи | Телеметрия сохраняется во внутренней памяти и передается после восстановления соединения |
| Передача данных | SMS и GPRS; TCP/UDP; протоколы EGTS, FLEX, FLEX 2.0 и FLEX 3.0; до 3 серверов |
| Защита данных | Шифрование AES128 при передаче на сервер |
| Настройка и обновление | NTC Configurator по USB, NTC Control по Bluetooth, удаленное обновление по GPRS и DRC |
| Ограничения подключения | Не поддерживаются тахографы, CAN-адаптеры, проводные термодатчики 1-Wire и оборудование RS-232 |
| Ограничения функций | Нет EcoDriving, фиксации ДТП, автоинформатора, фотокамеры и счетчиков пассажиропотока |
| Когда предлагать модель | Для базового 2G-мониторинга с одной SIM-картой, контролем топлива, охранными входами, четырьмя выходами, RS-485 и проводной громкой связью |
