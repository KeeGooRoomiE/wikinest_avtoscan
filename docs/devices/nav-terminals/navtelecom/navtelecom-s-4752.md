# СИГНАЛ S-4752

<img src="/docs/assets/devices/nav-terminals/navtelecom/navtelecom-s-4752.png" alt="СИГНАЛ S-4752" style="max-width:400px;height:auto">

СИГНАЛ S-4752 — аппаратура спутниковой навигации (АСН) для регулируемых проектов мониторинга пассажирского и грузового транспорта. Устройство определяет координаты, направление и скорость движения ТС по ГЛОНАСС/GPS, передает мониторинговую информацию по сетям 4G, 3G и 2G и сохраняет данные во внутренней памяти при отсутствии связи. Модель предназначена для оснащения ТС категорий М2, М3 и N, в том числе используемых для коммерческих перевозок пассажиров, перевозки детей, опасных, специальных, тяжеловесных и крупногабаритных грузов, а также отходов. На официальных ресурсах производителя S-4752 указан как сертифицированная АСН для выполнения требований ТР ТС 018/2011 и постановления Правительства РФ № 2216; также модель рекомендована для оснащения техники лесного комплекса в рамках постановления № 1378. В отличие от обычных терминалов серии СИГНАЛ, S-4752 оснащен LTE Cat. 4 с автоматическим переходом на 3G/2G, поддерживает две SIM-карты и SIM-chip, имеет встроенную АКБ 1000 мА·ч, microSD до 32 ГБ, шесть универсальных входов, четыре управляющих выхода и интерфейсы CAN, RS-232, RS-485 и 1-Wire. Терминал поддерживает кнопку «Сигнал бедствия», вызов диспетчера, двустороннюю голосовую связь через подключаемую гарнитуру, автоинформатор и получение фотоснимков от совместимой камеры. Bluetooth в данной модели отсутствует.

## Сервисное обслуживание

Настройка, диагностика и локальное обновление терминала выполняются через mini-USB с использованием NTC Configurator. Производитель поддерживает удаленное изменение настроек и обновление прошивки по каналу Internet, управление по SMS и Internet, а также работу в системе удаленного управления трекерами DRC. Для диагностики предусмотрены пользовательские и отладочные журналы GSM-модема, навигационного приемника и цифровых интерфейсов. Гарантийный ремонт выполняется изготовителем при соблюдении условий паспорта; ремонт, техническое обслуживание или модернизация неуполномоченными лицами являются основанием для отказа в гарантии. Подтвержденная информация о подменном фонде производителя в проверенных материалах отсутствует.

## Гарантия

Гарантийный срок на СИГНАЛ S-4752 составляет 36 месяцев с даты продажи. На встроенный аккумулятор и батарейку предоставляется отдельная гарантия сроком 12 месяцев, также с даты продажи. В течение гарантийного срока изготовитель выполняет бесплатный ремонт при соблюдении правил транспортирования, хранения, монтажа и эксплуатации. Для обращения требуется полностью и разборчиво заполненный паспорт с серийным номером, датой продажи, печатью продавца и подписью покупателя. Гарантия не распространяется на повреждения, возникшие из-за нарушения условий эксплуатации, несанкционированного ремонта или модернизации, нарушения гарантийных пломб и стикеров, а также эксплуатации совместно с неисправным либо неподтвержденным совместимым оборудованием.


## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Навигационные системы | ГЛОНАСС/GPS |
| Канал передачи данных | 4G LTE Cat. 4 с поддержкой 3G и 2G; TCP/UDP |
| SIM-карты | 2: внешняя mini-SIM и внутренняя nano-SIM; предусмотрена установка SIM-chip; поддерживаются многопрофильные SIM-карты |
| Антенны | Внешние ГЛОНАСС/GPS и GSM, разъемы SMA |
| Погрешность определения координат | 2,5 м в плане и 5 м по высоте; погрешность скорости — не более 0,1 м/с |
| Встроенная АКБ | Li-Po 3,7 В, 1000 мА·ч, с расширенным температурным диапазоном |
| Автономная работа | не менее 7 ч при передаче данных один раз в минуту; не менее 30 мин в режиме голосового соединения |
| Универсальные входы | 6: каждый может использоваться как аналоговый, дискретный или частотно-импульсный |
| Цифровые интерфейсы | RS-232, RS-485, 1-Wire и USB; Bluetooth отсутствует |
| CAN-интерфейс | 1; подключение к бортовой CAN-шине и работа с файлами декодирования |
| Управляющие выходы | 4 линии для подключения внешних исполнительных устройств |
| Внутренняя память | 32 МБ; до 236 000 телематических записей при размере пакета менее 127 байт |
| Карта памяти | microSD до 32 ГБ; не менее 2 000 000 записей телеметрии на каждый 1 ГБ |
| Акселерометр | встроенный; датчики слабого и сильного ударов, перемещения и наклона; диапазон измерения не менее ±24 g |
| Защита линий | защита питания и входных линий от длительных перенапряжений до 200 В |
| Степень защиты корпуса | IP54 |
| Рабочая температура со встроенной АКБ | от −40 до +85 °C; заряд аккумулятора допускается при температуре от 0 до +50 °C |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Регулируемые проекты АСН | Оснащение ТС категорий М2, М3 и N для выполнения требований ТР ТС 018/2011 и ПП РФ № 2216; модель также рекомендована для техники лесного комплекса по ПП РФ № 1378 |
| Мониторинг ТС | Координаты, высота, скорость, путевой угол, дата и время; передача VIN, категории и типа энергоносителя ТС |
| Тревожные функции | Подключение кнопок «Сигнал бедствия» и вызова диспетчера; тревожные сообщения отправляются вне очереди |
| Работа с дополнительным оборудованием | Получение данных от контактных, импульсных, аналоговых и цифровых датчиков по RS-232, RS-485, CAN и 1-Wire |
| Контроль топлива | Подключение аналоговых, частотно-импульсных и цифровых ДУТ через предусмотренные входы и интерфейсы |
| Данные CAN-шины | Один встроенный CAN-интерфейс для чтения параметров ТС и передачи выбранных данных на сервер мониторинга |
| Связь с водителем | Двусторонняя голосовая связь через подключаемую гарнитуру; поддержка кнопки вызова диспетчера |
| Автоинформатор | Автоматическое воспроизведение звуковых файлов с microSD при въезде и выезде из геозон |
| Фотоконтроль | Получение и передача фотоснимков от совместимой камеры по RS-232 или RS-485 |
| Удаленное управление | Четыре выхода для управления внешними исполнительными устройствами по SMS, GPRS-командам или заданным событиям |
| Работа при отсутствии связи | Мониторинговые данные сохраняются во внутренней памяти или на microSD и автоматически передаются после восстановления соединения |
| Настройка и обновление | NTC Configurator по USB, удаленное управление через DRC, SMS и Internet; поддерживается автоматическое обновление прошивки |
| Когда предлагать модель | Для проектов, где требуется сертифицированная АСН с 4G/3G/2G, двумя SIM-картами, внешними антеннами, тревожной кнопкой, голосовой связью, microSD и полным набором проводных интерфейсов. Для задач с беспроводными датчиками по Bluetooth модель не подходит |
