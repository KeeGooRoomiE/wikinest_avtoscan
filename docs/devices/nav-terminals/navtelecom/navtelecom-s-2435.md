# СМАРТ S-2435

<img src="/docs/assets/devices/nav-terminals/navtelecom/navtelecom-s-2435.png" alt="СМАРТ S-2435" style="max-width:400px;height:auto">

СМАРТ S-2435 — автомобильный GPS/ГЛОНАСС-терминал для спутникового мониторинга ТС, контроля местоположения, маршрутов и пробега, а также сбора данных с бортовых систем и подключённых датчиков. Устройство определяет координаты по ГЛОНАСС/GPS/BeiDou, передает телематические данные по GSM/GPRS и позволяет контролировать местоположение, маршруты, пробег, моточасы, расход топлива, температуру и события от подключенных датчиков. Модель поддерживает EcoDriving, Bluetooth, фиксацию события ДТП и удаленное управление внешними устройствами. В линейке S-243x S-2435 следует выбирать для проектов, где одновременно нужны две SIM-карты, встроенный CAN-интерфейс и полный набор интерфейсов RS-232, RS-485 и 1-Wire. Терминал оснащен встроенной АКБ 800 мА·ч; опционально предусмотрены два встроенных SIM-chip. В отличие от S-2433 модель имеет вторую SIM-карту и CAN-интерфейс, а в отличие от S-2437 сохраняет RS-232, но имеет один CAN-интерфейс. Корпус имеет степень защиты IP54; терминал предназначен для установки в защищенном месте внутри ТС. При эксплуатации со встроенной АКБ допустимый температурный диапазон составляет от −20 до +60 °C.
> Исполнение с внутренними антеннами выбирают для компактного стандартного монтажа в зоне устойчивого приёма — преимущественно для эксплуатации в городе, пригороде и на регулярных маршрутах с хорошим покрытием сотовой сети.

## Сервисное обслуживание

Настройка, управление, передача данных и диагностика терминала выполняются через USB с использованием программного обеспечения производителя; базовые операции также поддерживаются по Bluetooth и GSM. Техническая документация, прошивки и материалы для настройки размещаются на официальном сайте «Навтелеком». Ремонт, техническое обслуживание или модернизация, выполненные лицами, не уполномоченными производителем, приводят к утрате права на гарантийное обслуживание.

## Гарантия

Гарантийный срок на терминал составляет 3 года с даты продажи. На встроенный аккумулятор и батарейку предоставляется отдельная гарантия сроком 1 год, также исчисляемая с даты продажи. В течение гарантийного срока производитель выполняет бесплатный ремонт при соблюдении правил транспортирования, хранения, монтажа и эксплуатации. Право на гарантийное обслуживание утрачивается при несанкционированном ремонте, техническом обслуживании или модернизации, нарушении гарантийных пломб или стикеров, повреждениях от механических, химических, температурных и иных внешних воздействий, а также при эксплуатации совместно с неисправным оборудованием. Незначительные потертости, не влияющие на технические характеристики, не являются основанием для отказа в гарантии.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Навигационные системы | ГЛОНАСС/GPS/BeiDou |
| Канал передачи данных | GSM/GPRS (2G); Bluetooth 4.0 |
| SIM-карты | 2: внешняя miniSIM и внутренняя nanoSIM; опционально — 2 встроенных SIM-chip |
| Антенны | Встроенные GNSS и GSM |
| Погрешность определения координат | 2,5 м в плане; 5 м по высоте (50% CEP, статический режим) |
| Встроенная АКБ | Li-Po 3,7 В, 800 мА·ч; зарядка от USB |
| Универсальные входы | 3: аналоговые, дискретные и частотно-импульсные |
| Цифровые интерфейсы | RS-485, RS-232 и 1-Wire; USB для настройки и диагностики |
| CAN-интерфейс | 1 встроенный интерфейс CAN |
| Управляющие выходы | 2 линии типа «открытый коллектор» |
| Датчик движения | Встроенный 3-осевой акселерометр |
| Детекторы глушения | GSM и GNSS |
| Степень защиты и рабочая температура | IP54; со встроенной АКБ — от −20 до +60 °C |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Мониторинг ТС | Местоположение, перемещения, маршруты, пробег, стоянки и моточасы |
| Контроль топлива | Подключение аналоговых и частотных ДУТ; по RS-485 — до 16 цифровых LLS-совместимых ДУТ |
| Данные CAN-шины | Прямое подключение к одной CAN-шине ТС; для одновременной работы с несколькими шинами требуется внешний CAN-адаптер |
| Подключение оборудования | Тахографы, ДУТ, RFID-считыватели, MODBUS-устройства и другое оборудование по RS-232 или RS-485 |
| Контроль температуры | Подключение цифровых термодатчиков по 1-Wire |
| Идентификация водителя | Ключи TouchMemory и совместимые считыватели через 1-Wire |
| Стиль вождения | EcoDriving по данным встроенного акселерометра |
| Фиксация ДТП | По порогам ускорения, предусмотренным ГОСТ, или по индексу тяжести ДТП ASI |
| Удаленное управление | Два выхода для управления внешними устройствами, например блокировкой или сиреной |
| Тревожные события | Передача сообщений по событиям от входов и датчиков; приоритетная отправка тревожных сообщений |
| Работа при отсутствии связи | События сохраняются во внутренней энергонезависимой памяти и передаются после восстановления соединения |
| Когда предлагать модель | Для проектов с двумя SIM-картами, встроенным CAN, RS-232/RS-485/1-Wire, АКБ 800 мА·ч и подключением нескольких внешних устройств |
