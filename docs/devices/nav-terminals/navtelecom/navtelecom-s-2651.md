# СИГНАЛ S-2651

<img src="/docs/assets/devices/nav-terminals/navtelecom/navtelecom-s-2651.png" alt="СИГНАЛ S-2651" style="max-width:400px;height:auto">

СИГНАЛ S-2651 — автомобильный навигационный терминал для спутникового мониторинга ТС и подключения расширенного комплекса бортового оборудования. Устройство определяет координаты по ГЛОНАСС/GPS/BeiDou, передает телематические данные по GSM/GPRS (2G) и поддерживает две SIM-карты. В отличие от компактной серии СМАРТ модель рассчитана на проекты с большим количеством подключений: предусмотрены шесть универсальных входов, четыре управляющих выхода, интерфейсы CAN, RS-232, RS-485 и 1-Wire, а также microSD-карта объемом до 32 ГБ. Терминал работает с ДУТ, тахографами, CAN-шиной, датчиками температуры, RFID-считывателями, фотокамерой, микрофоном и динамиком. Встроенная АКБ 800 мА·ч обеспечивает не менее 6 часов автономной работы при передаче данных один раз в минуту. S-2651 следует выбирать для комплексных проектов, где достаточно одного CAN-интерфейса и сети 2G; для двух CAN-интерфейсов предназначен S-2653, для связи 3G — S-2654. Исполнение S-2651M отличается наличием датчика вскрытия корпуса. Корпус имеет степень защиты IP54; при работе со встроенной АКБ допустимый температурный диапазон составляет от −20 до +60 °C.
>Исполнение с внешними антеннами рекомендуется для междугородных перевозок, карьеров и разрезов, лесных участков, тайги и других удалённых территорий, а также для скрытой установки в экранированных местах, когда антенны необходимо вынести отдельно от терминала для улучшения приёма спутникового и сотового сигнала. При этом внешняя антенна улучшает качество приёма, но не обеспечивает связь там, где покрытие сети оператора полностью отсутствует.

## Сервисное обслуживание

Настройка, управление, передача данных и диагностика терминала выполняются через USB с использованием NTC Configurator; базовые операции поддерживаются по Bluetooth, USB и GSM через NTC Control. Производитель предусматривает удаленное обновление прошивки и изменение настроек по GPRS. Техническая документация, прошивки и материалы для настройки размещаются на официальном сайте «Навтелеком». Ремонт, техническое обслуживание или модернизация, выполненные лицами, не уполномоченными производителем, приводят к утрате права на гарантийное обслуживание.

## Гарантия

Гарантийный срок на терминал составляет 3 года с даты продажи. На встроенный аккумулятор и батарейку предоставляется отдельная гарантия сроком 1 год, также исчисляемая с даты продажи. В течение гарантийного срока производитель выполняет бесплатный ремонт при соблюдении правил транспортирования, хранения, монтажа и эксплуатации. Для гарантийного обращения требуется полностью и разборчиво заполненный паспорт изделия вместе с терминалом. Право на гарантийное обслуживание утрачивается при несанкционированном ремонте, техническом обслуживании или модернизации, нарушении пломб или стикеров, повреждениях от внешних воздействий и эксплуатации совместно с неисправным либо неподтвержденным совместимым оборудованием. Незначительные потертости, не влияющие на технические характеристики, не являются основанием для отказа в гарантии.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Навигационные системы | ГЛОНАСС/GPS/BeiDou |
| Канал передачи данных | GSM/GPRS (2G); Bluetooth 4.0; 3G не поддерживается |
| SIM-карты | 2: внешняя miniSIM и внутренняя nanoSIM |
| Антенны | Внешние GNSS и GSM, разъемы SMA |
| Погрешность определения координат | не более 2,5 м в плане; 5 м по высоте (50% CEP, статический режим) |
| Встроенная АКБ | Li-Po 3,7 В, 800 мА·ч |
| Автономная работа | не менее 6 ч при передаче данных по GSM один раз в минуту |
| Универсальные входы | 6: аналоговые, дискретные и частотно-импульсные |
| Цифровые интерфейсы | RS-485, RS-232, 1-Wire; USB для настройки и диагностики |
| CAN-интерфейс | 1; поддержка J1939 и пользовательских файлов декодирования |
| Управляющие выходы | 4 линии типа «открытый коллектор» |
| Внутренняя память | 8 МБ; до 51 700 телематических записей при пакете менее 127 байт |
| Карта памяти | microSD до 32 ГБ; запись телеметрии |
| Акселерометр | диапазон ±24 g; датчики ударов, перемещения и наклона |
| Детекторы глушения | GSM и GNSS |
| Степень защиты корпуса | IP54 |
| Рабочая температура со встроенной АКБ | от −20 до +60 °C |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Мониторинг ТС | Местоположение, маршруты, пробег, стоянки, скорость и моточасы |
| Контроль топлива | Аналоговые и частотные ДУТ; по RS-485 — до 16 цифровых ДУТ LLS |
| Данные CAN-шины | Встроенный CAN: J1939 и другие протоколы по файлам декодирования; J1708 через RS-485 |
| Работа с тахографами | Данные и события тахографов; выгрузка ddd-файлов для поддерживаемых моделей «Штрих», «Меркурий», VDO Continental и «Атол» |
| Ограничение для Атол Drive 5 | Для подключения требуется дополнительный преобразователь UART/RS-232 |
| Контроль температуры | До 8 цифровых термодатчиков по 1-Wire; поддержка совместимых Bluetooth-датчиков |
| Идентификация водителя | Ключи TouchMemory и Proximity-считыватели через 1-Wire |
| Стиль вождения и ДТП | EcoDriving; фиксация ДТП по ГОСТ или индексу ASI и отправка профиля события |
| Связь с водителем | Двусторонняя громкая связь через микрофон и динамик; поддержка Bluetooth-гарнитуры |
| Фотоконтроль | Подключение фотокамеры и отправка снимков на сервер по запросу |
| Пассажирский транспорт | Автоинформатор, дисплей водителя, маршрутные табло и счетчики пассажиропотока |
| Удаленное управление | Четыре выхода для сирены, внешней блокировки и других исполнительных устройств |
| Работа при отсутствии связи | Данные сохраняются во внутренней памяти или на microSD и передаются после восстановления соединения |
| Когда предлагать модель | Для комплексных 2G-проектов с двумя SIM-картами, одним CAN, RS-232/RS-485/1-Wire, тахографом, голосовой связью и большим числом датчиков |
