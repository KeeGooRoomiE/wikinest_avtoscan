# Omnicom Optim

## Установка пароля

```
*SETPWDID 336062883 8749#
```
> `336062883` — ID устройства, `8749` — новый пароль

## Подключение к серверу настройки

Когда в статусе отображается «Настройки» — отправить SMS:

```
*CONNECTSC#
```

> Можно отправить несколько раз подряд.

| Команда | Описание |
|---|---|
| `*CONNECTSC#` | Запрос на установку соединения терминала с сервером настройки |
