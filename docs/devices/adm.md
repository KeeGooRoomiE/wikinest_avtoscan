# ADM

## Поддерживаемые модели (удалённое конфигурирование)

`ADM007` `ADM007BLE` `ADM333` `ADM333BLE` `ADM333V2`

## Основные команды

| Команда | Описание |
|---|---|
| `ADD ME <пароль>` | Авторизация номера телефона (пароль по умолчанию — `0`) |
| `PASS X,Y` | Смена пароля (X — старый, Y — новый) |
| `STATUS` | Запрос текущего состояния терминала |
| `UPDATE` | Обновление прошивки |
| `REMOTE` | Удалённое подключение (доступно 1 час или до перезагрузки) |

## Смена сервера

```
SETHOST0 95.163.12.22,20701     (ADM)
SETHOST0 95.163.12.22,20303     (ADM100)
SETHOST0 5.159.100.97,20663
```

Через доменное имя:
```
SETHOST0 www.test.ru:1000
```

Проверка:
```
HOST0=www.test.ru:1000
```

## Заморозка координат (ADM100)

Если зажигание не подключено к аналоговому входу — настроить заморозку координат на стоянке.

В конфигураторе: **Настройки → Трек → «Заморозка координат на стоянке»**

Командой:
```
INSTATIC 15,1
```
