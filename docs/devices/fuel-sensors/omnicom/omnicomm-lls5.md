# OMNICOMM LLS 5

<img src="/docs/assets/devices/fuel-sensors/omnicom/omnicomm-lls5.png" alt="OMNICOMM LLS 5" style="max-width:400px;height:auto">

OMNICOMM LLS 5 — проводной емкостный датчик уровня топлива для контроля топлива в баках ТС и стационарных топливохранилищах. Датчик измеряет уровень и температуру, передает данные по RS-232 или RS-485 и поддерживает протоколы LLS и Modbus. Главное отличие модели — автоматическая корректировка калибровки при изменении состава и свойств топлива; заявленная погрешность ±0,5 % обеспечивается после калибровки в рабочем топливе либо при срабатывании автоподстройки после полной заправки бака. Доступны исполнения с измерительной частью 700, 1000, 1500, 2000, 2500 и 3000 мм; трубки можно подрезать под глубину конкретного бака с последующей повторной калибровкой. Модель предназначена для бензина, летнего и зимнего дизельного топлива и других жидких светлых нефтепродуктов. В паспорте LLS 5 отсутствует маркировка взрывозащиты: для объектов, где требуется сертифицированное взрывозащищенное исполнение, применяется Omnicomm LLS-Ex 5.

## Сервисное обслуживание

Настройка, диагностика и калибровка выполняются в программе Omnicomm Configurator через устройство настройки Omnicomm UNU-USB. Калибровку «Пустой/Полный» проводят в том виде топлива, в котором датчик будет работать; после подрезки измерительной части калибровку выполняют заново. Обновление встроенного программного обеспечения производится при подключении по RS-485. При неисправности оформляется рекламация изготовителю или через авторизованный сервисный центр. Система кожуха-пломбы и пломбы-стяжки позволяет выявлять несанкционированное вмешательство.

## Гарантия

Для Российской Федерации и стран СНГ гарантийный срок не ограничен при соблюдении условий транспортирования, хранения, монтажа и эксплуатации, установленных эксплуатационной документацией. Отдельная дата окончания гарантии и срок ее исчисления паспортом не устанавливаются. При отказе датчика оформляется рекламация изготовителю. Средний срок службы, указанный в паспорте, составляет 12 лет; это не ограничивает неограниченный гарантийный срок при выполнении гарантийных условий.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип датчика | Емкостный проводной датчик уровня топлива с измерением температуры и автоподстройкой к свойствам топлива |
| Измеряемая среда | Бензин, летнее и зимнее дизельное топливо и другие жидкие светлые нефтепродукты |
| Варианты длины измерительной части | 700, 1000, 1500, 2000, 2500 и 3000 мм |
| Погрешность измерения уровня | ±0,5 %; при калибровке в рабочем топливе либо после срабатывания автоподстройки при полной заправке |
| Интерфейсы | RS-232 и RS-485 |
| Протоколы передачи данных | LLS и Modbus; автоматическое определение протокола |
| Питание | от 7 до 75 В |
| Степень защиты корпуса | IP69K |
| Гальваническая изоляция | электрическая прочность не менее 2500 В |
| Рабочая температура | от −40 до +80 °C |
| Предельные температуры | −60 и +85 °C |
| Измерение температуры топлива | от −40 до +80 °C |
| Период измерения и выдача данных | измерение 1 раз в секунду; интервал автоматической выдачи от 1 до 255 секунд |
| Государственный реестр средств измерений | регистрационный номер 73764-19; межповерочный интервал — 2 года |
| Средний срок службы | 12 лет |
| Взрывозащищенное исполнение | у LLS 5 не заявлено; для зон с обязательной Ex-сертификацией применяется OMNICOMM LLS-Ex 5 |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Контроль топлива | Измерение уровня и температуры топлива для контроля остатка, заправок, сливов и расхода в системе мониторинга |
| Применение на ТС | Грузовой, пассажирский, сельскохозяйственный, строительный, карьерный, коммунальный и другой транспорт с баками для светлых нефтепродуктов |
| Автоподстройка к топливу | Корректирует калибровку при изменении диэлектрических свойств топлива; для срабатывания требуется полная заправка, непогруженный участок не более 100 мм от фланца |
| Подключение к терминалу | Прямое проводное подключение по RS-485 или RS-232; терминал должен поддерживать протокол LLS либо Modbus |
| Калибровка | Выполняется в рабочем топливе по точкам «Пустой/Полный» через Omnicomm Configurator и UNU-USB |
| Тарировка бака | Формирует соответствие кода датчика объему топлива; производитель рекомендует не менее 20 шагов заправки, возможна тарировка методом слива |
| Подрезка измерительной части | Длина после подрезки должна быть на 20 мм меньше глубины бака; REF-канал должен быть короче основной части не менее чем на 100 мм |
| Защита от вмешательства | Кожух-пломба закрывает крепеж, пломба-стяжка защищает разъем и позволяет выявить попытку вмешательства |
| Ограничения применения | Только жидкие светлые нефтепродукты. Модель не имеет заявленной маркировки взрывозащиты; для соответствующих зон выбирать LLS-Ex 5 |
| Когда предлагать модель | Когда требуется проводной ДУТ с погрешностью ±0,5 %, автоматической адаптацией к смене топлива, интерфейсами RS-485/RS-232 и усиленной защитой корпуса IP69K |
