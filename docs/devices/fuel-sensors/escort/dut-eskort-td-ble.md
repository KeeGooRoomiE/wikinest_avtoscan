# ЭСКОРТ TD-BLE

<img src="/docs/assets/1784255365586-dut-eskort-td-ble.png" alt="dut-eskort-td-ble.png" style="max-width:400px;height:auto">

**ЭСКОРТ TD-BLE** — автономный беспроводной ёмкостный датчик уровня топлива для измерения уровня светлых нефтепродуктов в баках ТС и стационарных ёмкостях. Датчик передаёт измеренные значения по Bluetooth по протоколу ЭСКОРТ BLE и не требует прокладки сигнального кабеля от бака к навигационному терминалу. Для ТС доступны исполнения с длиной измерительной части от 150 до 2500 мм, для стационарных ёмкостей — от 150 до 6000 мм. Измерительные трубки допускается подрезать или удлинять под конкретный бак; после изменения длины необходима повторная калибровка. Данные могут приниматься терминалом с поддержкой протокола ЭСКОРТ BLE либо передаваться в терминал по RS-485/RS-232 через адаптер BA-BLE. Модель имеет автономное питание, степень защиты IP69S и маркировку взрывозащиты 0Ex ia IIB T6 X. TD-BLE предназначен только для светлых нефтепродуктов и не применяется для электропроводящих жидкостей, включая воду.

## Сервисное обслуживание

Настройка, диагностика, калибровка и тарировка выполняются через приложение Escort Configurator на мобильном устройстве или через конфигуратор для ПК. После подрезки или удлинения измерительных трубок датчик необходимо откалибровать заново. При неисправности паспорт предписывает обращаться к изготовителю; разборка датчика пользователем запрещена. Съёмный элемент питания заменяется только вне взрывоопасной зоны, в антистатических перчатках; производитель указывает литий-тионилхлоридный элемент LS 14500.

## Гарантия

Гарантийный срок эксплуатации и хранения составляет 36 месяцев с момента отгрузки датчика предприятием-изготовителем; гарантийный срок хранения в составе этого периода — не более 12 месяцев. Гарантия действует при соблюдении условий эксплуатации, транспортирования и хранения. На дефекты, возникшие по вине потребителя из-за нарушения этих условий, гарантия не распространяется. Гарантийный срок и срок службы на элемент питания не устанавливаются. Заявленный срок службы — 7 лет.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип датчика | Ёмкостный беспроводной датчик уровня топлива с автономным питанием |
| Измеряемая среда | Светлые нефтепродукты в баках ТС и стационарных ёмкостях |
| Длина измерительной части для ТС | от 150 до 2500 мм |
| Длина измерительной части для стационарных ёмкостей | от 150 до 6000 мм |
| Погрешность измерения уровня | не более ±1 % от диапазона измерений |
| Интерфейс и протокол | Bluetooth Low Energy; протокол ЭСКОРТ BLE |
| Интервал передачи данных | 3 с для актуальных версий программного обеспечения |
| Дальность передачи | не менее 10 м при нормальных условиях, без помех и препятствий, при работе с BA-BLE |
| Питание | Съёмный литий-тионилхлоридный элемент питания, номинальное напряжение 3,6 В |
| Степень защиты корпуса | IP69S |
| Маркировка взрывозащиты | 0Ex ia IIB T6 X; искробезопасная электрическая цепь уровня ia |
| Допустимые взрывоопасные зоны | классы 0, 1 и 2; категории смесей IIA и IIB |
| Рабочая температура | от −45 до +50 °C |
| Предельный температурный диапазон | от −60 до +85 °C |
| Акселерометр | в отдельных исполнениях; диапазон измерения угла наклона 180°, погрешность ±5° |
| Госреестр средств измерений | регистрационный номер 82229-21; межповерочный интервал — 2 года |
| Срок службы датчика | 7 лет; на элемент питания срок службы и гарантия не распространяются |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Контроль топлива | Измерение уровня топлива для расчёта остатка, контроля заправок, сливов и расхода в системе мониторинга |
| Применение на ТС | Грузовой, пассажирский, сельскохозяйственный, строительный, коммунальный и другой транспорт со светлыми нефтепродуктами в баке |
| Стационарные ёмкости | Контроль уровня светлых нефтепродуктов в резервуарах, ёмкостях хранения и топливных баках стационарных объектов |
| Прямое подключение | Передача данных непосредственно на совместимый навигационный терминал, поддерживающий ЭСКОРТ BLE |
| Подключение через адаптер | Для терминалов без BLE используется BA-BLE с передачей данных по RS-485 или RS-232 по протоколу LLS |
| Настройка | Escort Configurator на смартфоне или ПК; поддерживается защита настроек паролем |
| Калибровка и тарировка | После изменения длины трубок требуется повторная калибровка; тарировочная таблица может храниться в системе мониторинга или в памяти датчика |
| Чёрный ящик | Локальное хранение данных с возможностью выгрузки через приложение за период до 30 дней |
| Применение во взрывоопасной среде | Допускается в зонах 0, 1 и 2 при соблюдении специальных условий эксплуатации по маркировке 0Ex ia IIB T6 X |
| Монтаж длинных исполнений | При длине измерительной части 2 м и более используется донный упор из монтажного комплекта |
| Обслуживание элемента питания | Замена выполняется вне взрывоопасной зоны; вскрытие и замена в опасной среде запрещены |
| Ограничения применения | Не предназначен для воды, молочных продуктов и других электропроводящих жидкостей |
