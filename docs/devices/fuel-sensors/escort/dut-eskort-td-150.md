# ЭСКОРТ ТД-150

<img src="/docs/assets/devices/escort/dut-eskort-td-150.jpg" style="max-width:400px;height:auto">

**ЭСКОРТ ТД-150** — проводной ёмкостный датчик уровня топлива для измерения уровня и температуры светлых нефтепродуктов (бензин, дизельное топливо, керосин) в баках ТС и стационарных ёмкостях. Основной способ подключения — цифровой интерфейс RS-485 с протоколом LLS; при необходимости датчик может выдавать частотный или аналоговый сигнал. Длина измерительной части может составлять от 150 до 6000 мм. Для применения на ТС утверждённый диапазон измерений составляет до 2500 мм, для стационарных ёмкостей — до 6000 мм. Измерительные трубки можно подрезать или удлинять под конкретный бак, после изменения длины требуется повторная калибровка. Датчик имеет степень защиты IP69S и маркировку взрывозащиты 0Ex ia IIB T6 Ga X (0Ex ia IIB T6 X). ТД-150 предназначен только для светлых нефтепродуктов и не применяется для воды, молочных продуктов и других электропроводящих жидкостей.

## Сервисное обслуживание

Настройка, диагностика, калибровка и тарировка проводного ТД-150 выполняются через программу Escort Configurator с подключением датчика к ПК через преобразователь USB–RS-485. После подрезки или удлинения измерительных трубок датчик необходимо откалибровать заново. Для измерительной части длиной 2 м и более применяется донный упор из монтажного комплекта. При неисправности паспорт предписывает обращаться к изготовителю; разборка датчика пользователем запрещена. При эксплуатации во взрывоопасной зоне подключение выполняется по требованиям искробезопасной цепи и с учётом схемы включения защитного резистора.

## Гарантия

Гарантийный срок эксплуатации составляет 5 лет с момента отгрузки датчика с предприятия-изготовителя. Гарантия действует при соблюдении условий эксплуатации, транспортирования и хранения. На дефекты, возникшие по вине потребителя вследствие нарушения этих условий, гарантия не распространяется. Заявленный срок службы датчика — 10 лет.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип датчика | Проводной ёмкостный датчик уровня топлива |
| Измеряемая среда | Светлые нефтепродукты в баках ТС и стационарных ёмкостях |
| Длина измерительной части | от 150 до 6000 мм; оговаривается при заказе |
| Утверждённый диапазон измерений | для ТС — до 2500 мм; для стационарных ёмкостей — до 6000 мм |
| Погрешность измерения уровня | не более ±1 % от диапазона измерений |
| Цифровой интерфейс и протокол | RS-485; протокол LLS, скорость 19200 бит/с |
| Частотный выход | от 300 до 4395 Гц |
| Аналоговый выход | от 0,2 до 9 В |
| Bluetooth | нет; беспроводная связь предусмотрена в отдельной модификации ТД-150 BLE |
| Степень защиты корпуса | IP69S |
| Маркировка взрывозащиты | 0Ex ia IIB T6 Ga X (0Ex ia IIB T6 X); искробезопасная цепь уровня ia |
| Допустимые взрывоопасные зоны | классы 0, 1 и 2; категории смесей IIA и IIB |
| Рабочая температура | от −45 до +50 °C |
| Предельный температурный диапазон | от −60 до +85 °C |
| Госреестр средств измерений | регистрационный номер 82228-21; межповерочный интервал — 2 года |
| Срок службы датчика | 10 лет |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Контроль топлива | Измерение уровня топлива для расчёта остатка, контроля заправок, сливов и расхода в системе мониторинга |
| Применение на ТС | Грузовой, пассажирский, сельскохозяйственный, строительный, коммунальный и другой транспорт со светлыми нефтепродуктами в баке |
| Стационарные ёмкости | Контроль уровня бензина, дизельного топлива, керосина, моторного масла и других светлых нефтепродуктов в резервуарах и ёмкостях хранения |
| Подключение по RS-485 | Основной вариант для навигационных терминалов с поддержкой протокола LLS; допускается работа нескольких адресуемых датчиков на одной линии |
| Частотный и аналоговый режимы | Используются, когда терминал не поддерживает RS-485, но имеет частотный или аналоговый вход |
| Настройка | Escort Configurator на ПК через преобразователь USB–RS-485; поддерживается изменение режима, диапазона, сетевого адреса и фильтрации |
| Калибровка и тарировка | После изменения длины трубок требуется повторная калибровка; тарировка выполняется под форму и объём конкретного бака |
| Термокомпенсация | Корректирует показания при изменении температуры топлива; отключение применяется только в отдельных случаях |
| Применение во взрывоопасной среде | Допускается в зонах 0, 1 и 2 при соблюдении специальных условий подключения искробезопасных цепей и требований к кабелю |
| Монтаж длинных исполнений | При длине измерительной части 2 м и более используется донный упор из монтажного комплекта |
| Ограничения применения | Не предназначен для воды, молочных продуктов и других электропроводящих жидкостей |
