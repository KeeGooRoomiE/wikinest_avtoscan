# Устройство съёма сигналов УСС — преобразователь для счётчиков жидкости

<img src="/docs/assets/1784520693993-E94ACEE8-E74C-4877-BC6A-9DEF15B29CEB.png" alt="E94ACEE8-E74C-4877-BC6A-9DEF15B29CEB.png" style="max-width:400px;height:auto">

**Назначение.** УСС преобразует угол поворота выходного вала измерителя объёма счётчика жидкости в электрические импульсы постоянного тока по двум каналам. Преобразование фотоэлектрическое: импульсы формируются дисковым обтюратором и щелевыми оптронами. Количество импульсов пропорционально углу поворота вала и числу зубьев ротора.

> **УСС не является самостоятельным счётчиком жидкости.** Устройство устанавливается в составе измерителя объёма и передаёт импульсы на вторичный прибор, контроллер или иную систему учёта. Выбор модификации определяется типом счётчика, количеством импульсов на оборот и требуемой ценой импульса.

## Модификации и применяемость

| Модификация | Применяемость по паспорту | Цена импульса | Имп./оборот | Рабочие пределы |
| --- | --- | --- | --- | --- |
| **УСС-Б-25** | ППВ-100-1,6; ППВ-100-6,4; ППВ-150-1,6; ППВ-150-6,4 | 1,0 л/имп. | 25 | 0…1000 имп./с; до 500 об/мин |
| **УСС-Б-70** | ППО-25-1,6 всех исполнений; колонки КГН; УИЖГ-20 | 0,01 л/имп. | 70 | 0…1000 имп./с; до 500 об/мин |
| **УСС-Б-50** | Топливораздаточные колонки | 0,01 л/имп. | 50 | 0…1000 имп./с; до 500 об/мин |
| **УСС-Б-25** | ППО-40-0,6 | 0,1 л/имп. | 25 | 0…1000 имп./с; до 500 об/мин |

## Гарантия

Гарантийный срок эксплуатации — 12 месяцев со дня отгрузки с предприятия-изготовителя. Гарантия действует при соблюдении условий транспортирования, хранения, монтажа и эксплуатации. Нормативный срок сохраняемости — 2 года, установленный срок службы — 6 лет, полный срок службы — 10 лет.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип преобразования | Фотоэлектрическое, при помощи дискового обтюратора и щелевых оптронов |
| Количество каналов | 2 независимых импульсных выходных канала |
| Номинальное питание | 12 В, допуск +10 / −15 % |
| Ток потребления | Не более 0,03 А |
| Максимальный ток каждого выхода | 30 мА по схеме подключения |
| Рабочий диапазон частоты | 0…1000 имп./с |
| Максимальная частота вращения вала | 500 об/мин |
| Длина штатного соединительного кабеля | 0,6…2,0 м |
| Максимальная длина линии до вторичного прибора | 1500 м |
| Масса | Не более 2,2 кг |
| Температура эксплуатации | От −40 до +50 °C |
| Относительная влажность | До 95 ± 3 % |

### Взрывозащита, защита корпуса и требования к подключению

| Параметр | Значение |
| --- | --- |
| Маркировка взрывозащиты | 2ExmIIT6 по предоставленному паспорту |
| Допустимые зоны установки | Взрывоопасные зоны класса 1 или 2 по нормативной базе, указанной в паспорте |
| Способ обеспечения взрывозащиты | Герметизация токопроводящих частей эпоксидным компаундом |
| Степень защиты оболочки | IP67 |
| Нагрев элементов электроники | Не более +80 °C |
| Заземление | При работе в составе изделий без собственного заземления УСС необходимо надёжно заземлить; сопротивление до доступных металлических частей — не более 0,1 Ом |
| Кабель внешних цепей | Медный многожильный кабель с изоляцией, стойкой к нефтепродуктам |
| Параметры линии связи | Индуктивность не более 0,2 мГн; ёмкость не более 0,1 мкФ; сопротивление не более 40 Ом |
| Пломбирование | Конструкция предусматривает пломбирование, исключающее доступ к элементам, влияющим на метрологические показатели |
| Ремонтопригодность | Преобразователь УСС при выходе из строя ремонту не подлежит; восстановление системы выполняется заменой устройства |

### Работа в составе счётчиков жидкости

| Параметр | Значение |
| --- | --- |
| Счётчики с овальными шестернями СЖ-ППО | Для ППО-25 указано исполнение УСС-Б-70, для ППО-40 — УСС-Б-25 |
| Винтовые счётчики СЖ-ППВ | Для ППВ-100 и ППВ-150 указано исполнение УСС-Б-25 с ценой импульса 1,0 л/имп. |
| Топливораздаточные колонки и установки | УСС-Б-50 для топливораздаточных колонок и УСС-Б-70 для колонок КГН и установки УИЖГ-20 |
| Вторичные приборы | Источником питания и приёмником информации может быть вторичный прибор (КУП, ЦБУ или аналогичный) |
| Надёжность | Средняя наработка на отказ — 100 000 ч; среднее время восстановления — 2 ч (заменой устройства) |
