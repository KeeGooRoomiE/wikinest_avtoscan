# Drive Base

<img src="/docs/assets/1784875892562-______________________.bmp" alt="______________________.bmp" style="max-width:400px;height:auto">

**НРД Drive Base** — цифровой тахограф с блоком СКЗИ (НКМ) для непрерывной и некорректируемой регистрации режима труда и отдыха водителей, а так же скорости, маршрута и пробега. Модель включена в действующий перечень ФБУ «Росавтотранс» под № 18 и является средством измерений утверждённого типа. Первичная поверка выполняется при производстве, межповерочный интервал — 7 лет. Тахограф устанавливается в гнездо 1 DIN, имеет два слота для карт водителей и встроенный термопринтер. Drive Base **не имеет класса взрывозащиты**, поэтому его нельзя устанавливать на автомобили, предназначенные для перевозки опасных грузов.

## Сервисное обслуживание

Ввод в эксплуатацию, настройка и техническое обслуживание выполняются мастерскими, включёнными в перечень ФБУ «Росавтотранс»; ремонт — мастерскими, авторизованными производителем. Плановое обслуживание проводится не реже одного раза в 3 года и включает замену блока СКЗИ (НКМ), замену батарейки и настройку тахографа. Калибровка и настройка требуются после ремонта или снятия тахографа, замены СКЗИ, изменения идентификационных данных ТС, шин, трансмиссии, датчика движения или его кабеля, а также при нарушении пломб. Блок СКЗИ и батарея размещены в отдельном отсеке за передней панелью; вскрытие отсека, замена компонентов и последующее опломбирование выполняются мастерской.  Для всей линейки тахографов НРД (Атол) **предусмотрен подменный фонд:** при сервисном случае клиенту может быть предоставлен подменный тахограф на время ремонта или замены, что помогает сократить простой транспортного средства.

## Гарантия

Производитель гарантирует работоспособность тахографа в течение 24 месяцев со дня ввода в эксплуатацию специалистом, имеющим право на техническое обслуживание, но не более 36 месяцев от даты производства изделия. Гарантия действует при соблюдении условий эксплуатации, транспортирования и хранения. Гарантийный случай подтверждается диагностикой авторизованного сервисного центра; для ремонта требуются тахограф, паспорт изделия и акт рекламации. Нарушение правил установки и подключения, несанкционированное вмешательство, механические и иные повреждения, а также повреждение или неправомерная замена пломб могут быть основанием для отказа в гарантийном обслуживании.

## Технические характеристики

### Метрологические характеристики

| Параметр | Значение |
| --- | --- |
| Погрешность измерения времени (60–86 400 с) | ±4 с |
| Погрешность измерения скорости по ГНСС (20–180 км/ч, PDOP ≤ 3) | ±2 км/ч |
| Погрешность измерения скорости по датчику движения (20–180 км/ч) | ±1 км/ч |
| Инструментальная погрешность определения координат по ГНСС (PDOP ≤ 3) | ±3 м |
| Погрешность определения координат по ГНСС (PDOP ≤ 3) | ±15 м |
| Погрешность измерения пробега (1–9 999 999,9 км) | ±1 % |
| Погрешность синхронизации времени с UTC(SU) по ГНСС | ±2 с |
| Погрешность синхронизации со шкалой времени блока СКЗИ по ГНСС | ±2 с |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Форм-фактор | 1 DIN (ISO 7736) |
| Слоты для карт | 2 (основной и сменный водитель) |
| Навигация | ГЛОНАСС / GPS |
| Интерфейсы | USB, разъём ABCD, CAN1, CAN2 |
| Выгрузка данных | Через USB-накопитель или карт-ридер; встроенный GSM-модуль и дистанционная выгрузка не предусмотрены |
| Принтер | Термопечатающий |
| СКЗИ (НКМ) | Замена каждые 3 года, а также при смене владельца и в иных предусмотренных законодательством случаях |
| Взрывозащита и опасные грузы | Класс взрывозащиты и Ex-маркировка не заявлены; модель не следует использовать на ТС, для которых требуется взрывозащищённое исполнение тахографа |
| Поверка | Первичная при производстве; межповерочный интервал 7 лет |
| Версия ПО | v.1.03.8002 и выше |
