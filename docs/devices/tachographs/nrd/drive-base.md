# Тахограф Drive Base

![НРД Drive Base](/docs/assets/devices/nrd/drive-base.jpeg)

НРД Drive Base — это цифровой тахограф с СКЗИ для контроля режима труда и отдыха водителей, фиксации скорости, пробега и данных эксплуатации ТС. Модель соответствует требованиям законодательства РФ, внесена в перечень моделей тахографов ФБУ «Росавтодранс» и является средством измерения с заводской поверкой на 7 лет.
