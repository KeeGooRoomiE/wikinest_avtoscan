# Тахограф Drive Base

![НРД Drive Base](/docs/assets/devices/nrd/drive-base.jpeg)

НРД Drive Base — это цифровой тахограф с СКЗИ для контроля режима труда и отдыха водителей, фиксации скорости, пробега и данных эксплуатации ТС. Модель соответствует требованиям законодательства РФ, внесена в перечень моделей тахографов ФБУ «Росавтотранс» и является средством измерения с заводской поверкой на 7 лет. Для всей линейки тахографов НРД (Атол) предусмотрен подменный фонд: при сервисном случае клиенту может быть предоставлен подменный тахограф на время ремонта или замены, что помогает сократить простой транспортного средства. При этом Drive Base не имеет класса взрывозащиты, поэтому его нельзя устанавливать на автомобили, предназначенные для перевозки опасных грузов.
При активизации в блок вносятся идентификационные данные конкретного транспортного средства и организации или физического лица, которому оно принадлежит. После этого НКМ становится связан с данным ТС и используется в его составе до окончания срока эксплуатации либо до появления оснований для замены.

**Срок эксплуатации:**
Для блока НКМ-2.11 срок эксплуатации составляет не более 3 лет с момента активизации при условии, что до активизации блок хранился не более одного года. После окончания срока эксплуатации или срока действия ключевой информации блок необходимо заменить.

**Гарантия:**
Для НКМ-2.11 производства ООО «Спецпроект-2» гарантийный срок составляет 12 месяцев с момента активизации и ввода в эксплуатацию, но не более 15 месяцев с даты производства. Точный срок и условия гарантии конкретного блока указываются в его формуляре. Гарантийный срок не следует путать со сроком эксплуатации: гарантия составляет один год, а штатный срок работы блока после активизации — до трех лет.


## Технические характеристики

### Метрологические характеристики


| Параметр                                                                                 | Значение |
| ------------------------------------------------------------------------------------------------ | ---------------- |
| Погрешность измерения времени                                         | ±4 с           |
| Погрешность измерения скорости по ГНСС (20–180 км/ч)       | ±2 км/ч      |
| Погрешность измерения скорости по датчику (20–180 км/ч) | ±1 км/ч      |
| Погрешность определения координат по ГНСС (PDOP ≤ 3)       | ±3 м           |
| Погрешность измерения пробега (1–9 999 999,9 км)                   | ±1%             |
| Погрешность синхронизации времени с UTC(SU) по ГНСС        | ±2 с           |

### Функциональные характеристики


| Параметр           | Значение                                        |
| -------------------------- | ------------------------------------------------------- |
| Форм-фактор      | 1 DIN (ISO 7736)                                        |
| Слоты для карт | 2 (основной и сменный водитель) |
| Навигация         | ГЛОНАСС / GPS                                    |
| Интерфейсы       | USB, разъём ABCD, CAN1, CAN2                      |
| Принтер             | Термопечатающий                          |
| СКЗИ (НКМ)          | Замена каждые 3 года                    |
| Версия ПО          | v.1.03.8002 и выше                                 |
