# Drive Smart

<img src="/docs/assets/devices/nrd/drive-smart.png" style="max-width:400px;height:auto">

**Drive Smart** — цифровой тахограф с блоком СКЗИ для непрерывной и некорректируемой регистрации скорости, маршрута, пройденного пути и РТиО водителей. Модель включена в действующий перечень ФБУ «Росавтотранс» под № 9 и является средством измерений утверждённого типа. Первичная поверка выполняется при производстве, межповерочный интервал — 7 лет. Тахограф устанавливается в гнездо 1 DIN, имеет два слота для карт водителей и встроенный термопринтер. Доступно исполнение Drive Smart GSM: при наличии связного модуля поддерживаются дистанционная выгрузка данных на сервер мониторинга и подключение дополнительного оборудования по RS-232/RS-485. Модель допускается к установке на ТС, перевозящие опасные грузы, при соблюдении специальных условий эксплуатации и настройки взрывобезопасности. 

## Сервисное обслуживание

Ввод в эксплуатацию, настройка и техническое обслуживание выполняются мастерскими, включёнными в перечень ФБУ «Росавтотранс»; ремонт — мастерскими, авторизованными производителем. Плановое обслуживание проводится не реже одного раза в 3 года и включает замену блока СКЗИ, замену батарейки и настройку тахографа. Внеочередная проверка и настройка требуются после ремонта или снятия тахографа, замены СКЗИ или батарейки, изменения идентификационных данных ТС, шин, трансмиссии, датчика движения или его кабеля, а также при нарушении пломб. Для всей линейки тахографов НРД (Атол) **предусмотрен подменный фонд:** при сервисном случае клиенту может быть предоставлен подменный тахограф на время ремонта или замены, что помогает сократить простой транспортного средства.

## Гарантия

Производитель гарантирует работоспособность тахографа в течение 24 месяцев со дня ввода в эксплуатацию специалистом, имеющим право на техническое обслуживание, но не более 36 месяцев от даты производства изделия. Гарантия действует при соблюдении условий эксплуатации, транспортирования и хранения. Гарантийный случай подтверждается диагностикой авторизованного сервисного центра; для ремонта требуются тахограф, паспорт изделия и акт рекламации. Нарушение правил установки и подключения, несанкционированное вмешательство, повреждение или неправомерная замена пломб могут быть основанием для отказа в гарантийном обслуживании.

## Технические характеристики

### Метрологические характеристики

| Параметр | Значение |
| --- | --- |
| Погрешность измерения времени (60–86 400 с) | ±4 с |
| Погрешность измерения скорости по ГНСС (20–180 км/ч, PDOP ≤ 3) | ±2 км/ч |
| Погрешность измерения скорости по датчику движения (20–180 км/ч) | ±2 км/ч |
| Инструментальная погрешность определения координат по ГНСС (PDOP ≤ 3) | ±3 м |
| Погрешность определения координат по ГНСС (PDOP ≤ 3) | ±15 м |
| Погрешность измерения пробега (1–9 999 999,9 км) | ±1 % |
| Погрешность синхронизации времени с UTC(SU) по ГНСС | ±2 с |
| Погрешность синхронизации со шкалой времени блока СКЗИ по ГНСС | ±2 с |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Форм-фактор | 1 DIN (ISO 7736) |
| Слоты для тахографических карт | 2 (основной и сменный водитель) |
| Навигация | ГНСС ГЛОНАСС/GPS |
| Основные интерфейсы | USB, разъём ABCD, CAN1, CAN2 |
| Исполнение Drive Smart GSM | Связной GSM-модуль; дистанционная выгрузка данных; RS-232 и RS-485 для подключения дополнительного оборудования |
| Принтер | Встроенный термопечатающий механизм |
| СКЗИ | Блок НКМ присутствует; замена не реже одного раза в 3 года, а также при смене владельца и в иных предусмотренных законодательством случаях |
| Взрывозащита и опасные грузы | Ex-маркировка 2Ex nA nC IIC T6 Gc X; для ТС с опасными грузами должен быть включён параметр «Взрывобезопасность» |
| Поверка | Первичная при производстве; межповерочный интервал 7 лет |
| Версия ПО | v.1.03.7999 и выше |
