# Меркурий ТА-001

<img src="/docs/assets/devices/tachographs/mercury.png" style="max-width:400px;height:auto">

**Меркурий ТА-001** — цифровой тахограф с блоком СКЗИ (НКМ) для непрерывной и некорректируемой регистрации скорости, маршрута, пройденного пути и РТиО водителей. Модель включена в действующий перечень ФБУ «Росавтотранс» под № 3 и является средством измерений утверждённого типа. Первичная поверка выполняется при выпуске, межповерочный интервал — 7 лет. Взрывопожарозащищённое исполнение допускается для ТС классов EX/II, EX/III, AT, FL и OX по ДОПОГ при соблюдении требований к монтажу.

## Сервисное обслуживание

Ввод в эксплуатацию и техническое обслуживание выполняются сервисным центром или мастерской, имеющими установленный допуск; гарантийный и послегарантийный ремонт — авторизованными сервисными центрами изготовителя. Плановое ТО проводится один раз в 3 года и включает проверку функционирования и точности показаний, а также настройку тахографа. Внеочередное ТО требуется после изменения окружности шин или характеристического коэффициента ТС, изменения VIN или регистрационного номера, ремонта или модернизации тахографа, замены блока СКЗИ и нарушения пломб. При признании тахографа непригодным во время ввода в эксплуатацию паспорт Производитель предусматривает **наличие подменного фонда:** при сервисном случае клиенту может быть предоставлен подменный тахограф на время ремонта или замены, что помогает сократить простой транспортного средства.
## Гарантия

Гарантийный срок на тахограф составляет 24 месяца с момента продажи или передачи первому покупателю — по дате продажи либо ввода в эксплуатацию, указанной в паспорте, но не более 36 месяцев с даты изготовления. На аккумуляторную батарею предоставляется гарантия 12 месяцев. Гарантийное обслуживание выполняет авторизованный сервисный центр при своевременном прохождении ТО и наличии отметок в паспорте. Гарантия не распространяется на регламентные работы, настройку, ввод в эксплуатацию и активацию, естественный износ, повреждения из-за неправильной эксплуатации, неоригинальных деталей или ремонта вне авторизованного центра.

## Технические характеристики

### Метрологические характеристики

| Параметр | Значение |
| --- | --- |
| Погрешность измерений интервала времени (60–86 400 с) | ±4 с |
| Погрешность измерений скорости по ГНСС (20–180 км/ч, PDOP ≤ 3) | ±2 км/ч |
| Инструментальная погрешность измерений скорости по датчику движения (20–180 км/ч) | ±2 км/ч |
| Инструментальная погрешность определения координат по ГНСС (PDOP ≤ 3) | ±3 м |
| Погрешность определения координат по ГНСС (PDOP ≤ 3) | ±15 м |
| Относительная погрешность измерений пройденного пути (1–9 999 999,9 км) | ±1 % |
| Погрешность синхронизации времени с UTC(SU) по ГНСС | ±2 с |
| Погрешность синхронизации со шкалой времени блока СКЗИ по ГНСС | ±2 с |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Применение | ТС категорий M2, M3, N2 и N3 для коммерческих перевозок пассажиров и грузов |
| Форм-фактор | Монтажная рамка 1 DIN |
| Слоты для тахографических карт | 2 (основной и сменный водитель) |
| Телематический терминал | Передача координат, скорости, путевого угла, времени и сигнала тревоги; GSM/GPRS — в соответствующих исполнениях |
| Работа при потере связи | Хранение не менее 150 000 точек координат и автоматическая передача после восстановления GPRS-соединения |
| Основные интерфейсы | RS-485, CAN, RS-232, USB; подключение внешних датчиков по RS-485 |
| Принтер | Встроенный термопринтер |
| СКЗИ | Блок СКЗИ входит в состав тахографа; замена выполняется с отзывом активации и последующей активизацией нового блока |
| Взрывозащита и опасные грузы | Взрывопожарозащищённое исполнение: 2Ex nA IIB T4 Gc X; применение на ТС классов EX/II, EX/III, AT, FL и OX по ДОПОГ |
| Поверка | Первичная при выпуске; межповерочный интервал 7 лет |
| Версия ПО | v.1.04.0138 и выше |
