# УОС «БАРС» — устройство ограничения скорости

<img src="/docs/assets/devices/additional/tachography/uos-bars-01-modul.jpg" style="max-width:400px;height:auto">

**УОС «БАРС»** — электронное устройство, предотвращающее движение транспортного средства выше заданного порога скорости за счёт ограничения подачи топлива. В исполнении Universal X v2.0 ограничение выполняется изменением выходного сигнала электронной педали акселератора. Устройство предназначено для категорий M2, M3, N2 и N3; диапазон настройки ограничения — 10–180 км/ч. Допускается установка на транспортные средства с дизельным двигателем либо бензиновым двигателем с электронной педалью; карбюраторные системы исключены.

## Исполнение зависит от типа педали газа

**Электронная педаль — модуль «БАРС» Universal X v2.0.** Подключается в разрыв выходных цепей педали и ЭБУ. Поддерживаются один или два аналоговых потенциометра, ШИМ-сигнал, а также комбинированная педаль ШИМ + аналоговый канал. Для ШИМ используется канал П1, для дополнительного аналогового сигнала — П2.

<img src="/docs/assets/devices/additional/tachography/uos-bars-03-mehanika.jpg" style="max-width:400px;height:auto">

**Механическая педаль — другое исполнение оборудования.** Для грузовых автомобилей со стандартной механической педалью применяется УОС с электромагнитным топливным клапаном, а не прямое подключение Universal X к педали. Клапан нормально закрытый, имеет регулируемый обводной канал; без питания обеспечивает максимальное ограничение подачи топлива, настраиваемое до минимальных оборотов двигателя независимо от степени нажатия педали. Состав оборудования и схема монтажа отличаются.

## Сервисное обслуживание и настройка

Модуль устанавливается в кабине под панелью приборов в любом рабочем положении, с исключением прямого несанкционированного доступа. Настройка выполняется через micro-USB фирменной программой: вводятся данные мастерской, источник скорости, W-фактор при импульсном источнике, скорость ограничения и тип электронной педали — аналоговый или ШИМ. После монтажа пломбируются корпус, датчик скорости, разъём тахографа, точки питания и подключения к цепям педали.

## Гарантия

Изготовитель гарантирует надёжную и безаварийную работу УОС при условии соблюдения покупателем правил транспортирования, хранения и монтажа, установленных инструкцией изготовителя. Гарантийный срок эксплуатации — 12 месяцев с момента продажи. Изготовитель в течение гарантийного срока своевременно и безвозмездно устраняет все скрытые неисправности, дефекты и несоответствия его техническим характеристикам, которые не были выявлены в процессе изготовления, контроля и испытания УОС, при условии соблюдения потребителем правил эксплуатации, транспортирования и хранения.

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Входной размах импульсов скорости | 3–40 В |
| Частота импульсов скорости | 1–2000 Гц |
| Погрешность измерения скорости по импульсному входу | Не более 1 % при частоте 2 кГц |
| Аналоговый режим, каналы П1 и П2 | Диапазон входного/выходного напряжения 0–12 В |
| Разрядность АЦП/ЦАП | 12 бит, 3 мВ/бит |
| ШИМ-режим, канал П1 | Напряжение 3,5–12 В; частота 100–1500 Гц |
| Дискретность измерения/генерации ШИМ | 153 нс |
| Максимальный ток нагрузки каналов П1 и П2 | 50 мА, предусмотрена защита от превышения |
| Вход «КУЗ» | Порог срабатывания 2 В; максимальное напряжение 40 В |
| Максимальное напряжение CAN | ±30 В |
| Источники информации о скорости | Датчик скорости ТС; импульсный выход тахографа B6/B7; CAN-шина; внешний источник по RS-485 |
| Принцип ограничения | При скорости ниже установленного порога устройство в реальном времени дублирует сигнал педали. При достижении порога выходные значения плавно изменяются в сторону сигналов холостого хода, записанных при первичном обучении. |
| Воздействие на тормозную систему | УОС не приводит в действие штатную тормозную систему транспортного средства. |
| Ограничение при поднятом кузове | Опционально скорость ограничивается до 15 км/ч при поступлении дискретного сигнала датчика поднятия кузова на вход «КУЗ». |
| Подключаемые цепи | micro-USB; CAN Low/High; вход датчика поднятия кузова; импульсный вход скорости; зажигание; масса; каналы педали П1/П2; RS-485 A/B; один зарезервированный вход. |
| Индикация | Зелёный светодиод — работа модуля; синий — активное ограничение; красный — ошибка. |
| Защита от вмешательства | Корпус после настройки закрывается саморазрушающейся пломбой, исключающей доступ к micro-USB. Изменение Vset фиксируется во FLASH-памяти с датой, временем и данными мастерской. |
| Требования к монтажу | Сечение соединительных проводов не менее 0,5 мм². Линия зажигания защищается внешним предохранителем не более 3 А и подключается непосредственно к замку зажигания; масса — непосредственно к кузову надёжным болтовым соединением. |
