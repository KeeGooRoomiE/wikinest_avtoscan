# УОС «Надёжный контроль 80» — устройство ограничения скорости

<img src="/assets/devices/additional/tachography/uos-nk80-01-ustroystvo.png" style="max-width:400px;height:auto">

**УОС «Надёжный контроль 80»** — управляемое электронное устройство, ограничивающее скорость транспортного средства регулированием подачи топлива в двигатель. При достижении заданного порога устройство воздействует на исполнительный механизм или цепи управления двигателем, после чего автомобиль перестаёт реагировать на дальнейшее нажатие педали и снижает скорость «накатом». Рабочая и стояночная тормозные системы не задействуются, двигатель не глушится. После снижения скорости ниже установленного значения управление тягой восстанавливается.

## Состав оборудования зависит от типа педали газа и двигателя

- **Электронная педаль.** Устанавливается основной блок УОС. Поддерживаются электронные педали с одним или двумя потенциометрическими каналами, а также педали с ШИМ-сигналами. Сигналы подключаются к входам POT1–POT4; дополнительный исполнительный механизм обычно не требуется.
- **Механическая педаль, дизель с ТНВД.** Помимо УОС требуется электромагнитный клапан ограничения подачи топлива. Он устанавливается в топливную магистраль после фильтра перед насосом и заказывается отдельно. Комплект, подключение и настройка отличаются от варианта с электронной педалью.
- **Механическая педаль, карбюратор.** Помимо УОС применяется шаговый двигатель с тросовым приводом в жёстком рукаве. При ограничении исполнительный механизм переводит дроссельную заслонку в закрытое положение независимо от положения педали. Это отдельная комплектация.

## Монтаж и настройка

Блок устанавливается внутри кабины на ровной поверхности; допускается вертикальное или горизонтальное положение. Питание подключается после замка зажигания. Сигнал скорости может поступать от тахографа, датчика скорости, датчика ABS, панели приборов либо CAN-шины. Настройка и программирование выполняются через внешний адаптер и программное обеспечение; входы IN1/IN2 и интерфейс RS-232 позволяют активировать дополнительные ограничения внешними устройствами.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип изделия | Интеллектуальное устройство ограничения скорости «Надёжный контроль 80» |
| Производитель | ООО «Жаворонок XXI ВЕК», производство на территории Российской Федерации |
| Назначение | Ограничение скорости путём регулирования подачи топлива без использования тормозных систем |
| Питание | Для бортовых сетей 12/24 В; заявленный производителем рабочий диапазон 10–40 В |
| Источники сигнала скорости | Тахограф, датчик скорости любого типа, ABS, датчик вращения вала КПП, панель приборов или CAN-шина |
| Типы педалей | Электронная; механическая с ТНВД; механическая с карбюратором — для каждого варианта различаются исполнительные механизмы и комплектация |
| Интерфейсы | CAN, открытый протокол RS-232, разъём внешнего устройства настройки |
| Защита питания | Встроенный предохранитель 2 А |

### Аппаратные и интерфейсные возможности

| Параметр | Значение |
| --- | --- |
| Микроконтроллер | 32-разрядный STMicroelectronics на ядре ARM Cortex-M3; частота 72 МГц |
| Память | 128 Кбайт памяти программы; 64 Кбайт оперативной памяти |
| Входы управления | Два аналоговых/дискретных входа; дополнительные входы IN1 и IN2 для активации отдельных ограничений |
| Выходы управления | Два универсальных выхода для исполнительных устройств и сигнализации |
| Электронная педаль | Входы POT1/POT2 для аналоговых сигналов; POT3/POT4 для ШИМ-сигналов |
| CAN | Встроенный CAN-контроллер; получение скорости и событий из CAN-линии, в том числе для сценариев ограничения |
| Встроенные узлы | Звуковой сигнализатор, акселерометр, каскад управления шаговым двигателем, регулируемый вход скорости |
| Индикация | Зелёный индикатор питания; красный индикатор достижения порога; SET-индикация типа педали; отдельный индикатор связи CAN |
| Управление порогом | Порог задаётся при программировании и может изменяться внешними сигналами или дополнительными кнопками |
| Функция «кик-даун» | Опциональное временное отключение ограничения на 30 секунд при резком полном нажатии педали |
| Количество сценариев | Производителем заявлена поддержка более 350 сценариев ограничения скорости |

### Функциональные возможности и примеры применения

| Параметр | Значение |
| --- | --- |
| Базовое ограничение скорости | Поддержание установленного верхнего порога скорости с автоматическим возвратом реакции педали после снижения скорости. |
| Специальное оборудование | Ограничение движения при поднятом кузове, неубранной КМУ, опущенном отвале, включённых щётках и других небезопасных положениях оборудования. |
| Геозоны и маршруты | Получение команды от системы мониторинга при въезде на территорию с низким лимитом либо при отклонении от разрешённого маршрута. |
| Состояние дороги и автомобиля | Автоматическое снижение скорости на неровной дороге; реакция на события TPMS, непристёгнутый ремень, движение задним ходом и присоединённый прицеп. |
| DMS/ADAS и алкотестеры | Интеграция с системами контроля водителя: ограничение или отключение педали по внешней команде при опасном сценарии. |
| Защита мониторинга | Ограничение скорости при обнаружении подавителя GPS/ГЛОНАСС по команде внешней системы. |
| Контроль холостого хода | Возможность остановки двигателя после заданного времени работы на холостом ходу с предварительным предупреждением. |
| Вилочные погрузчики и спецтехника | Ограничение до безопасной скорости, включая технику без электронной педали управления. |
| Перевозка детей и опасных грузов | На сайте производителя приведены сценарии ограничения автобусов с организованными группами детей и транспортных средств, перевозящих опасные грузы. |
| Сертификация | Утверждение типа по Правилам ООН № 89; соответствие ТР ТС 018/2011; испытания на электромагнитную совместимость по Правилам ООН № 10. |
| Правовая охрана | Исключительное право производства и распространения защищено патентом РФ № 193427. |
