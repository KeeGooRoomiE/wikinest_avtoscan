# VESA SYSTEM — датчики нагрузки на ось (рессорная подвеска)

<img src="/docs/assets/devices/axle-load/ressornaya-podveska.jpg" style="max-width:400px;height:auto">

**VESA SYSTEM** — бортовая система взвешивания для грузовых автомобилей, прицепов и специальной техники с рессорной подвеской. Система измеряет нагрузку на отдельные оси, полную массу транспортного средства и массу груза. Высокоточные датчики деформации устанавливаются непосредственно на мосты через приварные кронштейны; весовой контроллер преобразует их сигналы в весовые показатели и передаёт данные на дисплей в кабине и в систему мониторинга. По данным производителя, в зависимости от технического состояния подвески, условий эксплуатации и взвешивания погрешность обычно составляет 1–3 %. Для наилучшего результата необходимы корректная калибровка и исправное нормативное состояние подвески.

<img src="/docs/assets/devices/axle-load/ressornaya-podveska-schema.png" style="max-width:400px;height:auto">

## Сервисное обслуживание

В комплект поставки входит инструкция по установке. При эксплуатации необходимо контролировать состояние подвески, приварных кронштейнов, крепежа и кабельных жгутов; после работ, способных изменить деформационные характеристики мостов или подвески, рекомендуется повторная проверка показаний и калибровка. Настройка унифицированной конфигурации выполняется средствами EasyTuner; пользовательский интерфейс может открываться в браузере по Wi-Fi без доступа в интернет.

## Гарантия

Гарантия производителя на бортовые системы VESA — 12 месяцев. Заявленный ресурс эксплуатации системы — 5 лет.

## Технические характеристики

### Основные характеристики

| Параметр | Значение |
| --- | --- |
| Тип оборудования | Бортовая система взвешивания и контроля осевых нагрузок для транспортных средств с рессорной подвеской |
| Область применения | Грузовые автомобили, автопоезда, прицепы и специальная/сельскохозяйственная техника с рессорной подвеской |
| Измеряемые параметры | Нагрузка на отдельные оси, полная масса автомобиля или автопоезда, масса перевозимого груза |
| Принцип измерения | Измерение деформации мостов тензометрическими датчиками с последующим преобразованием сигналов весовым контроллером |
| Погрешность измерения | Обычно 1–3 %; в техническом каталоге для рессорной системы указано менее 3 % |
| Количество датчиков на мост | Один или два датчика деформации на каждый мост транспортного средства |
| Способ установки датчиков | Непосредственно на мост через приварные кронштейны; для сварочных работ используется монтажный шаблон |
| Тип датчиков | Аналоговые датчики деформации либо цифровые исполнения с интерфейсами CANBUS/RS485 |
| Датчик деформации моста | Питание 5–12 В DC; чувствительность 1 мВ/В; встроенная термокомпенсация |
| Количество каналов датчиков | До 6 аналоговых датчиков деформации на один блок (в исполнении VBox 1 — 5 каналов); точное исполнение контроллера уточняется при подборе комплектации |
| Питание системы | 24 В; для контроллера VBox 1 — 22–30 В DC, возможно исполнение под заказ на 12 В |
| Интерфейсы | 1×CAN J1939 и 1×RS485; данные могут передаваться на внешние устройства и терминал мониторинга |
| Степень защиты | IP67 |
| Температура эксплуатации | От −40 до +70 °C |
| Вывод информации водителю | Графический дисплей, цифровой индикатор, внешний бортовой компьютер или смартфон; подключение по CANBUS, RS-485 или Wi-Fi |
| Настройка | Универсальная конфигурационная модель EasyTuner; интерфейс в браузере по Wi-Fi без обязательного доступа в интернет |
| Удалённый мониторинг | Передача через внешний модуль спутникового мониторинга по цифровому интерфейсу |

### Функциональные характеристики

| Параметр | Значение |
| --- | --- |
| Основная комплектация | Весовой контроллер, датчики деформации и монтажный комплект |
| Монтажный комплект | Шаблон для сварки, приварные кронштейны, высокопрочные болты, предохранитель с колодкой, провода питания и цифровых интерфейсов, автомобильная гофра Schlemmer |
| Опциональное оборудование | Проводной индикатор загрузки для установки в панель кабины или герметичный бокс; также доступны графические и цветные дисплеи |
| Конфигурации по числу датчиков | В каталоге показаны варианты с 2, 3 и 4 датчиками: контроль массы груза/урожая, массы машины, нагрузки на три оси либо всех осей и полного автопоезда |
| Контроль в кабине и удалённо | Данные о нагрузках на оси, полной массе и весе груза отображаются водителю и могут передаваться в общую систему мониторинга транспорта |
| VESA ANALYTICA | Комплексный учёт работы каждой единицы, координаты и время погрузки/выгрузки, пройденное расстояние, затраченное время и топливо, перевезённый вес на каждом плече |
| События и уведомления | Контроль несанкционированных погрузок и выгрузок; при выходе параметров за нормативы возможны уведомления по электронной почте и СМС |
| Условия точного измерения | Исправная подвеска, соблюдение инструкции по взвешиванию и качественная калибровка; результат зависит от состояния транспортного средства и условий эксплуатации |
