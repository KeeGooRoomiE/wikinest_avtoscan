# Регламент переноса заявок под срочных клиентов

*2023 год.*

Данный регламент описывает процедуру:

1. Переноса заявок под срочных клиентов.
2. Определения сроков оформления заявок клиентов.

## Порядок переноса

Если расписание занято полностью и логист не может направить на данную установку свободного монтажника, в такой ситуации РОП должен совместно с ответственным логистом решить, какого клиента можно передвинуть, чтобы освободить время под срочного клиента.

Для решения этого вопроса РОПу предоставляется доступ к расписанию монтажных работ, и ему нужно самостоятельно согласовывать с клиентом перенос на более поздний срок.

При переносе клиентов и проведении срочных монтажей расписание с монтажником согласовывает логист.

## Оформление заявки без предварительной записи

Если клиента по предварительному согласованию с МОП поставили в расписание без оформленной заявки, то по данному клиенту необходимо оформить заявку и предоставить все необходимые данные до 15:00 дня, предшествующего дню установки оборудования.

В случае если заявка и необходимые данные всё-таки не будут предоставлены, логист может отменить данного клиента и поставить вместо него того, по которому предоставлены все необходимые данные.
