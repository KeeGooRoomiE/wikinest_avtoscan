# Форма заявки в техподдержку (сверка показаний)

Если клиент просит проверить работоспособность сразу нескольких ТС (расхождение показаний пробега или расхода топлива с путевыми листами/чеками заправок), специалисту ТП нужно попросить клиента заполнить одну из форм ниже — по одному транспортному средству на строку, с указанием даты события.

Каждая форма — таблица со столбцами: **№, г/н ТС, дата события, показание Wialon, показание клиента (путевой лист / чек), разница, % погрешности** (столбцы разницы и погрешности считаются в таблице автоматически).

## Форма по пробегу

Столбцы: пробег по Wialon / пробег по путевому листу.

📥 [Скачать шаблон (xlsx)](/docs/assets/processes/support/forma-zayavki-tp.xlsx)

## Форма по топливу

Столбцы: заправка по Wialon / заправка по чеку.

📥 [Скачать шаблон (xlsx)](/docs/assets/processes/support/forma-zayavki-tp.xlsx)

*Обе формы находятся в одном файле на разных листах: «Заявка по пробегу» и «Заявка по топливу».*
