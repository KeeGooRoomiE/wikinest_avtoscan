# Фотофиксация работ по установке оборудования

**Фотофиксация работ по установке оборудования навигации и тахографии.**

Данная мера обусловлена необходимостью периодически проверять качество
выполненных монтажных работ различного оборудования специалистами отдела
сервисного обслуживания ООО «Автоскан-Сервис», формирования
доказательной базы в виде фотоматериалов при возникновении вопросов
качества выполнения вышеуказанных работ, а так же для предварительного
анализа неисправности при сборе информации при подготовке к выполнению
ремонтных заявок. Фотофиксации подвергаются:

\- работы по установке любого оборудования (навигационный терминал,
тахограф, датчик уровня топлива и т.д.)

\- вмешательство в работу оборудования (со стороны клиента), намеренный
вывод из строя оборудования (со стороны клиента), смонтированное
оборудование не по регламенту (другой компанией);

\- работы по опломбировке (все установленные пломбы), замена или
установка СИМ карты;

\- работы по калибровке ТС (на одном фото должно быть отображено:
параметры шины и свидетельство о регистрации ТС)

Для решения этой задачи специалисту Отдела сервисного обслуживания
предоставляется смартфон, для проведения фото фиксации и передачи
фотоматериалов на адрес электронной почты <foto@navi42.ru>.
(<foto96@navi42.ru> –для ЕКБ)

Требования к содержанию фотоотчёта при выполнении работ с навигационным
терминалом:

1.  Общий вид ТС с государственным номером ТС. Номер должен полностью
    читаться, по фото возможно идентифицировать марку и модель ТС;

2.  Общий план установленного АТ с возможностью корректно определить
    место установки;

3.  Фото АТ с полностью читаемыми данными: Модель АТ, S/N, IMEI;

4.  Фото Sim карты с данными ICCID;

5.  Фото места установки GPS антенны (при наличии);

6.  Фото установки GSM антенны (при наличии);

7.  Точки подключения питания АТ с фиксацией пломб и возможностью
    определить место подключения;

8.  Фото подключения АТ к дополнительному оборудованию (CAN-шина, ДУТ,
    Датчик температуры, дополнительные входы/выходы) с фиксацией пломб и
    возможностью определить место подключения;

9.  Фото приборной панели при запущенном ДВС (при возможности запуска
    ДВС) ДО/ПОСЛЕ выполнения работ. (требуется как доказательная база,
    что после выполнения работ на ТС не появилось никаких ошибок, в т.ч
    новых при их наличии до начала выполнения работ);

10. Фото/скрин настроек АТ в конфигураторе;

11. Фото/скрин передаваемой Телеметрии;

Требования к содержанию фотоотчёта при выполнении работ с Тахографом:

1.  Общий вид ТС с государственным номером ТС. Номер должен полностью
    читаться, по фото возможно идентифицировать марку и модель ТС;

2.  Фото «шильдика» тахографа;

3.  Точки подключения питания тахографа с фиксацией пломб и возможностью
    определить место подключения;

4.  Фото подключения тахографа к дополнительному оборудованию (датчик
    скорости, CAN-шина, дополнительные входы/выходы) с фиксацией пломб и
    возможностью определить место подключения;

5.  Фото приборной панели при запущенном ДВС (при возможности запуска
    ДВС) ДО/ПОСЛЕ выполнения работ. (требуется как доказательная база,
    что после выполнения работ на ТС не появилось никаких ошибок, в т.ч.
    новых при их наличии до начала выполнения работ);

6.  Общий план установленного тахографа с возможностью корректно
    определить место установки;

7.  Фото показаний пробега на дисплее тахографа после выполнения
    калибровки;

8.  Фото показаний одометра на приборной панели ТС;

9.  Фото колеса с читаемой информацией о размере шины;

10. Фото калибровочной наклейки;

11. Фото датчика скорости/фишки датчика скорости, при выполнении работ,
    связанных с установкой/демонтажем/заменой датчика скорости;

12. Фото установленного спидометра, при выполнении работ, связанных с
    установкой/демонтажем/заменой спидометра;

13. Фото блока СКЗИ, установленного в тахограф, при выполнении работ,
    связанных с заменой/диагностикой блока СКЗИ;

14. Фото/скрин экрана с результатами диагностики блока СКЗИ с помощью
    АРМ-диагностики, при выполнении работ по диагностике блока СКЗИ;

15. Фото существующих ошибок на экране тахографа (при их наличии), при
    выполнении работ, связанных с диагностикой тахографа/блока СКЗИ;

Требования к содержанию фотоотчёта при выполнении работ с дополнительным
оборудованием (ДУТ, LLD. CAN-log, TH-BLE, TD-Base т.д.:

1.  Общий вид ТС с государственным номером ТС. Номер должен полностью
    читаться, по фото возможно идентифицировать марку и модель ТС;

2.  Общий план установленного оборудования с возможностью корректно
    определить место и корректность установки;

3.  Фото оборудования с полностью читаемыми данными: Модель
    оборудования, S/N, mac-адрес;

4.  Фото подключения дополнительного оборудования к АТ с фотофиксацией
    соединений, пломб;

5.  Фото/скрин настроек в конфигураторе устанавливаемого оборудования;

6.  Фото/скрин передаваемых данных в телеметрии АТ;

При установке нескольких устройств фотоотчёт формируется для каждого
устройства отдельно.

В теме отправленного электронного письма техник должен указать номер
Заказа-наряда (например, «2-1234»). В случае, когда заказ-наряд выдан
без номера, техник указывает в теме письма гос. номер ТС и наименование
Заказчика (например, а822аа, КРУ).

Электронное письмо с материалами должно быть отправлено установщиком до
момента сдачи Заказ-наряда.

Заполняя Заказ-наряд, установщик обязан в графе «Наименование работ»
указать тот факт, что фотоматериалы были отправлены на электронный адрес
в ТП (достаточно одного слова «фото»)

Примечание.

В случаях, когда установщик выезжает на работы по заявке Технической
поддержки, регламент фиксации следующий:

**Фото 1.** Общий вид кабины тс с гос. номером ТС .

**Фото 2.**Причина неисправности оборудования (повреждение проводки,
сломана антенна и т.д.)

**Фото 3.** Результат проделанной работы, согласно регламента .

Последовательность фотографий обязательна.
