# Определение ставок оплаты для монтажников

**Регламент определения ставок оплаты для монтажников**

При продаже оборудования клиентам, зачастую требуется оказать услугу по
установки, настройки этого оборудования. Услуги по установки и настройки
оборудования, оказывают монтажники из сервисного отдела. За такие услуги
монтажникам начисляется сдельная заработная плата. При возникновении
потребности в оказании новых услуг для клиентов, есть необходимость в
определении ставок для оплаты монтажникам за эти услуги.

В большинстве случаев ставка определятся из правила, 10% от стоимости
оказанной услуги. Но есть исключения, в которых ставку необходимо
определить в индивидуальном порядке, например:

1.  Стоимость услуги искусственно занижена для клиента, на основаниях
    конкурентной цены, или по другим причинам.

2.  Дорогостоящие основные услуги, с большим объемом работ включающие в
    себя много этапов, и времени на исполнение. При определении ставок
    по правилу 10% для таких услуг, полученная стоимость ставки
    несоизмерима с объемом проделанной работы. Пример, установка датчика
    уровня топлива с тарировкой бака до 1т. стоит 4000р. 10% от этой
    стоимости несоизмеримо с объемом работ. По этому, ставка за такую
    услугу равна 800р.

При определении ставок для таких услуг, необходимо просчитать объем
работ, и сопоставить его с объемом работ подобных действующих ставок,
далее принять стоимость ставки близко к известной, равной по сложности
ставки.

В нашей компании принята система четырех категорий монтажников. При
расчете и создании новой ставки, ее стоимость определяется для базовой
второй категории. Для остальных категорий ставка определяется согласно
коэффициентам принятых в регламенте категорий. После определения новой
ставки, необходимо согласовать ее стоимость с директором.
