# Инструкция по формированию ежемесячного отчёта по Абонентской плате

*В папку руководителя отдела продаж. 02.04.2024.*

## Цель

Данная инструкция разработана для того, чтобы РОП мог самостоятельно, без ошибок и по установленным правилам формировать отчёт о финансовых показателях отдела за месяц по абонентской плате. Эта инструкция не содержит пошаговых действий для формирования отчёта, а акцентирует внимание на тех местах, в которых наиболее часто возникают ошибки при формировании отчёта.

## Показатели, которые необходимо отобразить в отчёте

1. **Дебиторская задолженность по АО.** Данные берём из отчёта 1С «Ведомость расчётов с клиентами ОП». После формирования отчёта за выбранный период, в разделе «Конечный остаток» указываем «Долг клиента» по каждой организации.

2. **Кредиторская задолженность по АО.** Данные берём из отчёта 1С «Ведомость расчётов с клиентами ОП». После формирования отчёта за выбранный период, в разделе «Конечный остаток» указываем «Наш долг» по каждой организации.

3. **Оплаченная задолженность по АО.** Данные берём из отчёта 1С «Ведомость расчётов с клиентами ОП». После формирования отчёта за выбранный период, в разделе «Оплачено» указываем поступления ДС по каждой организации. Если в течение месяца были «перепривязки» или возврат ДС клиенту, то отклонения будут отображаться в разделе «Зачёт аванса» и необходимо учитывать их при подсчёте поступлений (как со знаком «+», так и со знаком «−»).

4. **Количество объектов в Wialon.** Указываем то количество объектов, за которое нам выставил ООО «Автоскан-Сервис» (Новосибирск) по итогу месяца, либо количество объектов (ТС), которое находится у нас в Wialon. Также необходимо указать количество объектов, находящихся на «ГлонассСофт». Эти данные можно уточнить у Титоренко А.В. или Лимакова С.А.

5. **Количество объектов в 1С с начислением АП.** Чтобы указать количество объектов (ТС) со статусом «активен» на конец месяца, переходим в отчёт 1С «Анализ активности телематического оборудования — Мальцева» и берём итоговую цифру в столбце «Активно конечный остаток».

6. **Начисление АП СДС.** Указываем количество объектов по подразделениям СДС (АО «Черниговец», ООО «ШУМ», ООО «СИГД»), которые НЕ находятся на обслуживании в ПО Wialon, но оплачивают абонентскую плату. Точное количество по каждому подразделению можно посмотреть в УПД (акт выполненных работ). Как правило количество ТС меняется очень редко (не чаще 2 раз в год).

После заполнения всех полей отчёт отправляется главному бухгалтеру до 10 числа текущего месяца.
